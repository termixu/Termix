---

## 🧠 ФИЛОСОФИЯ АРХИТЕКТУРЫ

**Простота, масштабируемость, кот.**

Платформа Termix Academy — это не монолит. Это набор микросервисов, объединённых вокруг кота-проводника.

Три главных принципа:
- **Termix везде.** Кот присутствует в каждом компоненте: в интерфейсе, в уведомлениях, в системе баджей.
- **API-first.** Всё общение между фронтендом и бэкендом идёт через REST API.
- **Геймификация встроена.** Баджи, жетоны, хвост Termix'а — не отдельный модуль, а ядро платформы.

---

## 🧩 ВЫСОКОУРОВНЕВАЯ АРХИТЕКТУРА

Пользователь (браузер, Telegram, мобильное приложение) общается с платформой через три интерфейса:

- **Web-приложение** — React SPA.
- **Telegram-бот** — Python, aiogram.
- **Мобильное приложение Termius** — React Native.

Все три интерфейса обращаются к единому REST API, которое работает на FastAPI. API общается с базой данных PostgreSQL. Фоновые задачи (отправка уведомлений, подсчёт статистики) обрабатываются через Celery с Redis как брокером сообщений. Мониторинг — Prometheus и Grafana.

Все компоненты упакованы в Docker-контейнеры и управляются через Docker Compose. CI/CD — GitHub Actions.

---

## 🖥️ FRONTEND — REACT SPA

Веб-приложение, с которым взаимодействует ученик или наставник через браузер.

**Основные страницы:**
- Главная (дашборд с прогрессом).
- Курсы (каталог, карточка курса, урок).
- Профиль (баджи, жетоны, ранг, хвост Termix'а).
- Сообщество (Discord-виджет, чат).
- Терминал (встроенный эмулятор для командных курсов).

**Ключевые фичи:**
- Анимированный Termix в углу экрана (реагирует на прогресс).
- Встроенный Spline-плеер для 3D-сцен.
- Встроенный Blockly-редактор для визуального программирования.
- Code-песочница (Monaco Editor, как в VS Code).

**Стек:** React, React Router, Tailwind CSS, Monaco Editor, Spline WebGL.

---

## ⚙️ BACKEND — FASTAPI

REST API, которое обслуживает все запросы от фронтенда, бота и мобильного приложения.

**Основные модули:**
- Пользователи (регистрация, авторизация, роли).
- Курсы (каталог, уроки, прогресс).
- Баджи и достижения (выдача, коллекция).
- Термикс AI (чат с котом, подсказки, генерация мемов).
- Уведомления (email, Telegram, push).
- Платежи (интеграция с платёжным шлюзом).

**Ключевые фичи:**
- JWT-авторизация.
- Role-based access (ученик, наставник, админ).
- API-документация через Swagger (автоматически).
- WebSocket для реального времени (чат, уведомления).

**Стек:** FastAPI, SQLAlchemy, Alembic (миграции), Celery, Redis, Prometheus.

---

## 🗄️ DATABASE — POSTGRESQL

Основная база данных платформы.

**Основные таблицы:**
- users (ученики, наставники, админы).
- courses, modules, lessons (структура курсов).
- enrollments (записи на курсы).
- user_progress (прогресс по урокам).
- badges, achievements, user_badges (система баджей).
- submissions (отправленные задания).
- termix_logs (логи взаимодействия с котом).
- transactions (платежи и подписки).

**Ключевые фичи:**
- Полнотекстовый поиск по курсам.
- Аналитика (кто сколько прошёл, где застрял).
- Миграции через Alembic.

---

## 🤖 TELEGRAM-БОТ

Бот Termix Assistant — внешний интерфейс платформы для лёгкого доступа.

**Основные команды:**
- `/start` — приветствие и регистрация.
- `/progress` — текущий прогресс ученика.
- `/lesson` — пройти следующий урок.
- `/termix` — спросить у кота-ИИ.
- `/battle` — код-батл с другим учеником.
- `/achievements` — коллекция баджей.

**Ключевые фичи:**
- Интеграция с основным API.
- Отправка мем-шпаргалок по запросу.
- Ежедневные задания (с утра приходит задача дня).
- Кнопка «SOS Кэп» — связь с наставником.

**Стек:** Python, aiogram, интеграция с FastAPI через HTTP.

---

## 📱 МОБИЛЬНОЕ ПРИЛОЖЕНИЕ TERMIUS

Приложение, которое объединяет сериал, терминал и квесты.

**Основные разделы:**
- Эпизоды сериала «Termix: Core Keeper».
- Встроенный терминал для практики команд Linux.
- Квесты (повтори команду из серии, найди ошибку, спаси сервер).
- Прогресс (хвост Termix'а растёт и меняет цвет).

**Ключевые фичи:**
- Офлайн-доступ к пройденным урокам.
- Push-уведомления о новых эпизодах.
- Синхронизация прогресса с основной платформой.

**Стек:** React Native, интеграция с FastAPI через HTTP.

## 🐳 DOCKER И ДЕПЛОЙ

Все компоненты упакованы в Docker-контейнеры.

**docker-compose.yml включает:**
- web (React-фронтенд, порт 3000).
- api (FastAPI-бэкенд, порт 8000).
- db (PostgreSQL, порт 5432).
- redis (брокер для Celery, порт 6379).
- celery_worker (фоновые задачи).
- bot (Telegram-бот).

**CI/CD через GitHub Actions:**
- При пуше в main: тесты → сборка Docker-образов → деплой на VPS.
- При пуше в dev: тесты → сборка → деплой на staging.

## 📊 МОНИТОРИНГ

- Prometheus собирает метрики (запросы в секунду, время ответа, ошибки).
- Grafana отображает дашборды (здоровье системы, активность учеников).
- Алерты в Telegram при падении сервера или росте ошибок.

---

## 🔒 БЕЗОПАСНОСТЬ

- JWT-токены с ограниченным временем жизни.
- HTTPS (SSL-сертификат через Let's Encrypt).
- CORS настроен только для доверенных доменов.
- Пароли хешируются (bcrypt).
- API-ключи хранятся в переменных окружения (`.env`).
- Rate limiting на API (100 запросов в минуту на пользователя).
