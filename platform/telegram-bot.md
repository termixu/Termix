## 🧠 ФИЛОСОФИЯ БОТА

**Учиться можно прямо в Telegram, без браузера.**

Telegram-бот Termix Academy — это полноценный интерфейс платформы. Ребёнок может:

- Проходить уроки в формате диалога с Termix'ом.
- Получать мем-шпаргалки по командам.
- Выполнять ежедневные задания.
- Сражаться в код-батлах с другими учениками.
- Отслеживать свой прогресс и баджи.

Бот не заменяет платформу. Он дополняет её. Всё, что сделано в боте, синхронизируется с основным профилем через API.

---

## 🎯 ОСНОВНЫЕ КОМАНДЫ

- `/start` — Приветствие от Termix'а и выбор роли (ученик или наставник).
- `/menu` — Главное меню с кнопками.
- `/progress` — Мой прогресс, баджи, ранг.
- `/lesson` — Пройти следующий урок (текст, картинки, мемы).
- `/code` — Проверить код (бот выполняет Python или JS в песочнице).
- `/termix` — Спросить Termix'а (AI-чат с котом).
- `/daily` — Ежедневное задание (плюс опыт).
- `/battle` — Код-батл с другим учеником (PVP).
- `/achievements` — Мои достижения и баджи.
- `/profile` — Профиль пирата (кастомизация кота).
- `/ship` — Команда (группа учеников).
- `/captain` — Связь с наставником Артёмом (SOS).

---

## 🏗️ АРХИТЕКТУРА БОТА

Бот построен на библиотеке aiogram 3.x и состоит из нескольких модулей:

- **Обработчики команд** — реакции на `/start`, `/help`, `/menu`.
- **Обработчики сообщений** — реакции на ключевые слова и команды без слеша.
- **Клавиатуры** — Reply-кнопки и Inline-кнопки для навигации.
- **Интеграция с API** — все данные о прогрессе, курсах и баджах берутся с основного бэкенда.
- **AI-чат** — общение с Termix'ом через OpenAI API или аналог.
- **Песочница кода** — выполнение Python и JS в изолированной среде.
- **Код-батлы** — PvP-система для соревнований между учениками.

## 📋 ОСНОВНЫЕ ФИЧИ

### Квестовая система уроков

Уроки приходят как диалог с Termix'ом, а не как простыня текста. После теории — интерактивный квиз с кнопками.

Ученик нажимает `/lesson`. Termix присылает первый фрагмент урока. Ученик отвечает на вопрос. Termix хвалит или поправляет. В конце урока — бадж и жетоны.

### Мем-шпаргалки

По запросу ученик может получить шпаргалку. Пишет `git push` — бот присылает картинку с Termix'ом и попугаем. Пишет `404` — бот присылает Termix'а, кричащего на табличку «404».

### Ежедневный попутный ветер

В 9:00 бот присылает задачку дня. Решил — получил опыт и монеты. Не решил — ничего страшного, завтра новая.

### Код-батлы (PvP)

Два ученика получают одну задачу. Например: «Напиши функцию pirate_greet(name), которая возвращает "Arrr, name!"». Кто решил быстрее — получает бадж «Дуэлянт» и жетоны. Вызов через `/battle @username`.

### AI-чат с Termix'ом

Ученик пишет `/termix` и вопрос. Например: «Почему мой Python-код не работает?». Termix отвечает в своём стиле: «Бип! Давай посмотрю. Покажи код, юнга!». Используется OpenAI API или аналог с системным промптом Termix'а.

### Связь с наставником

Кнопка «SOS Кэп» (`/captain`) отправляет вопрос напрямую Артёму. Используется для сложных вопросов и поддержки.

## 🔄 ИНТЕГРАЦИЯ С ПЛАТФОРМОЙ

Бот общается с основным API платформы через HTTP-запросы:

- **Авторизация:** привязка Telegram ID к аккаунту на платформе.
- **Прогресс:** бот отправляет результаты уроков, платформа обновляет прогресс.
- **Баджи:** платформа проверяет условия и выдаёт баджи.
- **Уведомления:** бот получает уведомления от платформы и пересылает ученику.

## 🛠️ ТЕХНИЧЕСКИЙ СТЕК

- **Язык:** Python 3.11+.
