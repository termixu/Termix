## 📂 ОПИСАНИЕ ПАПОК

### `docs/` — Документация и философия

- **manifesto.md** — Манифест Termix Academy: миссия, ценности, принципы.
- **philosophy.md** — Философия обучения: почему мышление важнее синтаксиса, геймификация.
- **branding.md** — Брендбук: логотип, цвета (`#00ff41`, `#0a0a0a`, `#ffd700`), шрифты, стиль.
- **pricing.md** — Модели ценообразования: разовая оплата, подписка, пакеты, B2B.
- **roadmap.md** — Roadmap проекта: что сделано, что в процессе, что запланировано.

### `termix/` — Всё о коте Термиксе

- **termix-character.md** — Полное описание персонажа: внешность, глаза, шляпа, крюк, хвост, одежда.
- **termix-tail.md** — Кибер-хвост: 8 цветов, поведение, анимации, интеграция с системой «Сбои Ядра».
- **termix-phrases.md** — Полный сборник фраз: для сериала, приложения, уроков, стикеров.
- **termix-visual-style.md** — High-end 8bit стиль: полная и короткая версии промпта.
- **termix-wallpapers/** — Обои с Термиксом (PNG, разные разрешения).
- **termix-stickers/** — Стикерпаки (PNG, 512x512).

### `programs/` — Полный список программ

- **programs-by-age.md** — Программы с разбивкой по возрастным группам (5–7, 6–8, 7–9, 8–10, 9–11, 10–12, 11–13, 12–14, 13–15, 14–16 лет).
- **programs-by-type.md** — Программы с разбивкой по типам: Графика и 3D, Видео и Анимация, Программирование, Веб, Игры, Инструменты, Мобильная разработка, Монетизация, AI, Офисные.

### `professions/` — Профессии

- **professions-index.md** — Индекс всех профессий с ссылками.
- **20 файлов профессий** — Каждая профессия: описание, путь в Termix Academy, инструменты, зарплаты, проекты для портфолио.

### `courses/` — Курсы

- **courses-index.md** — Индекс всех курсов с ссылками и возрастными рекомендациями.
- **10 папок по направлениям:**
  - `thinking/` — Мышление программиста.
  - `computer-literacy/` — Microsoft Office, PowerPoint.
  - `visual-programming/` — ПиктоМир, Scratch Jr, Scratch, Google Blockly, Kodu, Delightex, Hopscotch, Clickteam Fusion, App Inventor, Thunkable.
  - `game-development/` — Minecraft Edu, Roblox Studio, Godot, Unreal Engine 4.
  - `web-development/` — HTML, CSS, Figma.
  - `programming-languages/` — JS Robot, JavaScript, Python, PHP, Java, C#, C++.
  - `3d-and-design/` — LEGO Digital Designer, Tinkercad, Paint.NET, Piskel, Spline, GIMP, SketchUp, VoxEdit, Blender, Adobe Photoshop.
  - `video-and-animation/` — Clideo, Animaker, CapCut.
  - `tools-and-infrastructure/` — VS Code, Terminal, Git, Linux, Docker, GitHub Actions, Jupyter, JupyterLab, Termius, React.
  - `data-and-ai/` — SQL, AI-промптинг, Copilot.
  - `business/` — Steam Workshop, FAB, Фриланс, Telegram Bot API.

### `systems/` — Системы Termix Academy

- **core-failures.md** — Система предупреждений «Сбои Ядра»: Помеха, Сбой, Авария, восстановление, журнал наставника.
- **homework-missions.md** — Система домашних заданий (Миссии): три уровня, пиратские свитки, секретные коды.

### `platform/` — Платформа Termix Academy

- **architecture.md** — Архитектура платформы: фронтенд, бэкенд, база данных, API.
- **database-schema.md** — Схема базы данных: таблицы, связи, SQL-запросы.
- **telegram-bot.md** — Телеграм-бот: команды, функции, интеграция с платформой.
- **termix-app.md** — Приложение Termius: концепция, сериал, терминал, квесты, монетизация.

### `assets/` — Материалы

- **badges/** — Баджи курсов (PNG, 256x256).
- **memes/** — Мем-шпаргалки (PNG, готовые к печати).
- **prompts/** — Готовые промпты: стандартный стиль, весёлый стиль, Termix с технологиями.

### `promo/` — Маркетинг и продвижение

- **termix-series.md** — Концепция сериала «Termix: Core Keeper»: 10 серий, сценарий, промпты.
- **landing-page.md** — Текст для лендинга: заголовки, преимущества, призывы.
- **social-media.md** — Посты для Instagram, TikTok, Telegram: шаблоны, хештеги.
- **parent-letter.md** — Письмо родителям: почему IT, почему Termix Academy, цены.

### `archive/` — Архив

- Старые версии файлов, черновики, устаревшие материалы.
