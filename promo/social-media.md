## 🎯 СТРАТЕГИЯ

**Платформы:** Instagram (Reels + посты), TikTok, Telegram-канал.

**Частота:** 3–4 поста в неделю.

**Формат контента (правило 3-2-1):**
- 3 образовательных поста (команды, концепции, лайфхаки).
- 2 развлекательных (мемы с Termix'ом, рилсы).
- 1 продающий (кейсы учеников, анонсы курсов).

**Тон:** Весёлый, пиратский, с «Бип!» и «Мяу!». Никакого официоза.

**Визуал:** Все изображения в стиле high-end 8bit. Termix в каждом втором посте.

## 📝 ШАБЛОНЫ ПОСТОВ

### Тип 1: Образовательный пост (Команда недели)

**Формат:** Картинка с Termix'ом + текст.

**Заголовок:** 🦾 Команда недели: `git push`

**Текст:**
```

Бип! Каждый понедельник Termix рассказывает про одну команду.

Сегодня: git push 🦜

Что делает: отправляет твой код на GitHub.
Пиратская аналогия: ты отправляешь почтового попугая с картой сокровищ в главный штаб.

Как запомнить: push = толкать. Толкаешь код НА сервер.

А как часто ты пушишь? 👇

#termix #git #codingforkids #программирование

```

**Изображение:** Termix с попугаем, попугай улетает в небо.

---

### Тип 2: Мем с Termix'ом

**Формат:** Картинка с Termix'ом + смешная подпись.

**Текст:**
```

Когда написал 100 строк кода, а он не работает...

Termix: «Бип! Проверь точку с запятой. Это всегда точка с запятой.»

Ставь ❤️ если было такое

#termix #programmingmemes #кодер #мемы

```

**Изображение:** Termix с грустным лицом, хвост жёлтый, сидит перед монитором с ошибкой.

```
Теперь Вася хочет стать разработчиком игр. А его мама не верит, что это сделал её сын.

Хочешь так же? Записывайся на пробное занятие. Ссылка в шапке профиля 🚀

#termix #кейс #python #программированиедлядетей

```

**Изображение:** Termix с гордым видом, рядом — скриншот бота.

### Тип 5: Рилс (короткое видео)

**Формат:** Вертикальное видео 15–30 секунд.

**Идеи для рилсов:**

- Termix печатает на клавиатуре, экран мигает зелёным. Текст: «Когда наконец-то починил баг». Музыка: эпичная.
- Termix открывает системный блок и смотрит внутрь. Текст: «Я просто хотел почистить от пыли... А нашёл там флешку с 2015 года».
- Termix с красным хвостом. Текст: «Когда код не работает уже 3 часа». Музыка: драматичная.
- Termix с зелёным хвостом. Текст: «Когда запушил и всё работает с первого раза». Музыка: победная.
- Termix пьёт чай из Java-кружки. Текст: «Java-разработчики по утрам». Музыка: спокойная.
