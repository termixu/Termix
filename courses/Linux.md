## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: Linux → чёрный экран → сложно → «я сломаю систему».

Подход Termix Academy: Linux → подводная лодка → ты — капитан → «пеленгую, погружаюсь, всплываю».

Linux — это операционная система, на которой работает весь интернет. Серверы, облака, суперкомпьютеры, Raspberry Pi — всё на Linux. Termix учит детей управлять системой через терминал: пользователи, права, процессы, пакеты, сеть. К концу курса ребёнок может поднять свой сайт на настоящем сервере.

## 📖 ЗАНЯТИЕ 1: ПОГРУЖЕНИЕ

### 🎯 Цель

Понять, что такое Linux, освоить базовые команды и структуру файловой системы.

### ⏱️ Тайминг (75–90 минут)

- 0–5 мин: Termix: «Бип! Добро пожаловать на борт подводной лодки! Linux — это твоя система жизнеобеспечения. Погнали!»
- 5–15 мин: Что такое Linux. Ubuntu, Debian, серверы. Виртуальная машина или WSL.
- 15–25 мин: Файловая система. `/` — корень. `/home` — каюты. `/etc` — приборы. `/var` — склады.
- 25–45 мин: Базовые команды: `pwd`, `ls`, `cd`, `mkdir`, `touch`, `rm`, `cp`, `mv`.
- 45–55 мин: `man` — документация. `--help` — справка.
- 55–60 мин: Итоги. Бадж «Погружение».

### 🛠️ Практика

```bash
pwd                          # где я?
ls -la /                     # что в корне системы?
cd /home                     # перейти в домашнюю папку
mkdir submarine              # создать папку
cd submarine
touch logbook.txt            # создать файл
echo "Погружение начато" > logbook.txt
cat logbook.txt
man ls                       # документация по ls
