---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: HTML → теги → `<div>` → `</div>` → скучно → «это просто текст с галочками».

Подход Termix Academy: HTML → каркас корабля → заголовок — нос, секции — палубы, ссылки — канаты → «я построил сайт!».

HTML — это скелет каждого сайта в интернете. Без него не работает ничего. Termix учит детей собирать веб-страницы как пиратский корабль: каркас, обшивка, переборки. Каждый тег — деталь судна. К концу курса ребёнок сверстает многостраничный сайт Termix Academy.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Понимать структуру HTML-документа.
- Использовать базовые теги: заголовки, параграфы, ссылки, изображения.
- Создавать списки и таблицы.
- Строить формы.
- Понимать семантику: header, main, footer, nav, section.
- Встраивать видео и аудио.
- Связывать страницы в многостраничный сайт.
- Понимать, как HTML связан с CSS и JS.

---

## 🗺️ КАРТА КУРСА

- **Занятие 1 — «Киль и шпангоуты».** Структура документа. `<html>`, `<head>`, `<body>`. Первая страница.
- **Занятие 2 — «Палубы и переборки».** Заголовки h1-h6. Параграфы. Переносы. Горизонтальные линии.
- **Занятие 3 — «Канаты и якоря».** Ссылки `<a>`. Абсолютные и относительные. Якоря внутри страницы.
- **Занятие 4 — «Сундуки и флаги».** Изображения `<img>`. Форматы. Атрибуты alt, width, height.
- **Занятие 5 — «Списки и таблицы».** ul, ol, li. table, tr, td, th. Инвентаризация.
- **Занятие 6 — «Формы: Бортовой журнал».** form, input, textarea, button, select.
- **Занятие 7 — «Семантика: Генеральный план».** header, nav, main, section, article, footer.
- **Занятие 8 — «Флот: Многостраничный сайт».** Свободный проект. Связывание страниц.

## 📖 ЗАНЯТИЕ 1: КИЛЬ И ШПАНГОУТЫ

### 🎯 Цель

Понять структуру HTML-документа, создать первую страницу.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Бип! Добро пожаловать на верфь! HTML — это каркас нашего корабля. Киль — `<html>`, шпангоуты — `<head>` и `<body>`. Погнали!»
- 5–15 мин: Что такое HTML. Теги. Открывающие и закрывающие. Атрибуты.
- 15–25 мин: Структура: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`.
- 25–45 мин: Первая страница. Заголовок страницы (`<title>`). Текст на странице.
- 45–55 мин: Просмотр в браузере. DevTools (F12) — посмотреть на свой HTML.
- 55–60 мин: Итоги. Бадж «Киль».

### 🛠️ Практика

Ребёнок создаёт файл `index.html`:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пиратский Штаб</title>
</head>
<body>
    Бип! Привет, юнга! Это мой первый сайт.
</body>
</html>
```

Открывает в браузере. Нажимает F12 → Elements. Видит свой HTML.

📖 ЗАНЯТИЕ 2: ПАЛУБЫ И ПЕРЕБОРКИ

🎯 Цель

Освоить заголовки, параграфы и базовое форматирование.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Палубы! Заголовки — это уровни корабля. h1 — капитанский мостик. h6 — трюм. Расставляем по порядку!»
· 5–15 мин: Заголовки <h1> — <h6>. Только один h1 на страницу.
· 15–30 мин: Параграфы <p>. Перенос строки <br>. Горизонтальная линия <hr>.
· 30–50 мин: Форматирование: <strong>, <em>, <b>, <i>, <u>.
· 50–55 мин: Практика. Создаём разметку страницы.
· 55–60 мин: Итоги. Бадж «Палуба».

🛠️ Практика

```html
<body>
    <h1>Termix Academy</h1>
    <p><strong>Бип!</strong> Добро пожаловать в <em>пиратскую</em> академию программирования.</p>
    <hr>
    <h2>Наши курсы</h2>
    <p>Мы учим юнг:</p>
    <h3>Git</h3>
    <p>Пиратский контроль версий.<br>С котом и попугаем.</p>
    <h3>Python</h3>
    <p>Заклинания для змейки.</p>
</body>
```

Игра «Расставь по рангу». Наставник диктует: «Капитан (h1), Штурман (h2), Боцман (h3), Юнга (h4)». Дети создают заголовки.

🏠 Миссия

· 🥉 Юнга: Создать заголовки h1, h2, h3 и параграф.
· 🥈 Боцман: Добавить форматирование (жирный, курсив) и линию.
· 🥇 Штурман: Сверстать структуру страницы «О капитане» с заголовками и параграфами.

📖 ЗАНЯТИЕ 3: КАНАТЫ И ЯКОРЯ

🎯 Цель

Освоить ссылки.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Канаты связывают корабль с берегом. Ссылки связывают страницы. Бросаем якорь!»
· 5–15 мин: Тег <a>. Атрибут href.
· 15–30 мин: Абсолютные ссылки (https://...). Относительные ссылки (page.html).
· 30–50 мин: Якоря: #section. Навигация внутри страницы.
· 50–55 мин: Атрибут target="_blank" — открыть в новой вкладке.
· 55–60 мин: Итоги. Бадж «Штурман».

🛠️ Практика

```html
<h1>Пиратская Навигация</h1>

<h2>Внешние ссылки</h2>
<p><a href="https://github.com">GitHub — главный штаб</a></p>
<p><a href="https://termix.academy" target="_blank">Termix Academy (в новой вкладке)</a></p>

<h2>Внутренняя навигация</h2>
<p><a href="#crew">К экипажу</a></p>
<p><a href="#treasures">К сокровищам</a></p>

<h2 id="crew">Экипаж</h2>
<p>Termix, Вася, Петя.</p>

<h2 id="treasures">Сокровища</h2>
<p>Золото, рыба, карта.</p>

<p><a href="#">Наверх</a></p>
```

Квест «Собери карту». Наставник: «Сделай 3 якоря на странице и ссылки на них сверху». Кто первый — жетон.

🏠 Миссия

📖 ЗАНЯТИЕ 5: СПИСКИ И ТАБЛИЦЫ

🎯 Цель

Освоить списки и таблицы.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Инвентаризация! Списки — это перечень добычи. Таблицы — учёт золота. Считаем!»
· 5–15 мин: <ul> (маркированный), <ol> (нумерованный), <li>.
· 15–30 мин: <table>, <tr>, <td>, <th>. Структура таблицы.
· 30–45 мин: Объединение ячеек: rowspan, colspan.
· 45–55 мин: Практика. Таблица экипажа.
· 55–60 мин: Итоги. Бадж «Инвентаризатор».

🛠️ Практика

```html
<h2>Снаряжение</h2>
<ul>
    <li>Карта сокровищ</li>
    <li>Компас</li>
    <li>Сабля</li>
</ul>

<h2>План</h2>
<ol>
    <li>Проснуться</li>
    <li>Поднять паруса</li>
    <li>Найти сокровище</li>
</ol>

<h2>Экипаж</h2>
<table border="1">
    <tr>
        <th>Имя</th>
        <th>Звание</th>
        <th>Золото</th>
    </tr>
    <tr>
        <td>Termix</td>
        <td>Капитан</td>
        <td>999</td>
    </tr>
    <tr>
        <td>Вася</td>
        <td>Юнга</td>
        <td>10</td>
    </tr>
</table>
```

📖 ЗАНЯТИЕ 6: ФОРМЫ — БОРТОВОЙ ЖУРНАЛ

🎯 Цель

Освоить создание форм.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Бортовой журнал! Формы собирают данные. Кто на борту? Сколько золота? Записываем!»
· 5–15 мин: <form>. Атрибуты action и method.
· 15–30 мин: <input>: text, password, email, number, checkbox, radio.
· 30–50 мин: <textarea>, <button>, <select>, <option>.
· 50–55 мин: <label>. Связывание с input.
· 55–60 мин: Итоги. Бадж «Летописец».

🛠️ Практика

```html
<h2>Регистрация в экипаже</h2>
<form>
    <label for="name">Имя пирата:</label>
    <input type="text" id="name" placeholder="Введи имя"><br><br>

    <label for="rank">Звание:</label>
    <select id="rank">
        <option>Юнга</option>
        <option>Боцман</option>
        <option>Штурман</option>
        <option>Капитан</option>
    </select><br><br>

    <label for="gold">Золото:</label>
    <input type="number" id="gold" min="0" max="9999"><br><br>

    <input type="checkbox" id="agree">
    <label for="agree">Согласен с пиратским кодексом</label><br><br>

    <button type="submit">Взять на борт!</button>
</form>
```

🎮 Игровая механика

📖 ЗАНЯТИЕ 7: СЕМАНТИКА — ГЕНЕРАЛЬНЫЙ ПЛАН

🎯 Цель

Освоить семантические теги HTML5.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Генеральный план! <div> — это просто ящик. А <header> — это капитанский мостик. Семантика — порядок!»
· 5–15 мин: <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>.
· 15–30 мин: Замена <div> на семантические теги.
· 30–50 мин: Собираем страницу по плану.
· 50–55 мин: Просмотр в DevTools. Структура стала понятнее.
· 55–60 мин: Итоги. Бадж «Архитектор».

🛠️ Практика

```html
<body>
    <header>
        <h1>Termix Academy</h1>
        <nav>
            <a href="#home">Главная</a>
            <a href="#courses">Курсы</a>
            <a href="#crew">Экипаж</a>
        </nav>
    </header>

    <main>
        <section id="home">
            <h2>Добро пожаловать!</h2>
            <p>Бип! Здесь учат кодить с пиратским котом.</p>
        </section>

        <section id="courses">
            <h2>Курсы</h2>
            <article>
                <h3>HTML</h3>
                <p>Каркас корабля.</p>
            </article>
            <article>
                <h3>Python</h3>
                <p>Заклинания для змейки.</p>
            </article>
        </section>
    </main>

    <footer>
        <p>&copy; 2025 Termix Academy. Все права защищены.</p>
    </footer>
</body>
```
