> **Версия:** 1.0
>
> **Возраст:** 10–14 лет

>

---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: VS Code → сотня кнопок → «я ничего не понимаю» → ребёнок пишет в Блокноте.

Подход Termix Academy: VS Code → капитанский пульт → расширения → хоткеи → «я управляю кодом как профи».

Visual Studio Code — это главный инструмент современного разработчика. Бесплатный, мощный, с кучей расширений. Termix учит детей не бояться редактора, а настраивать его под себя: темы, шрифты, сниппеты, форматирование, встроенный терминал. К концу курса ребёнок чувствует себя капитаном за пультом управления кодом.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Уверенно ориентироваться в интерфейсе VS Code.
- Открывать папки и проекты (не по одному файлу).
- Пользоваться встроенным терминалом.
- Устанавливать расширения.
- Использовать горячие клавиши.
- Форматировать код автоматически.
- Работать с поиском и заменой.
- Использовать сниппеты для быстрого написания кода.
- Настраивать тему и шрифт под себя.

## 📖 ЗАНЯТИЕ 1: ПЕРВОЕ ЗНАКОМСТВО

### 🎯 Цель

Установить VS Code, освоить интерфейс, открыть папку, создать первый файл.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Бип! VS Code — это твой капитанский пульт. Здесь ты управляешь кодом. Погнали настраивать!»
- 5–15 мин: Установка. Скачиваем с code.visualstudio.com. Запускаем.
- 15–25 мин: Интерфейс. Боковая панель, редактор, строка состояния.
- 25–40 мин: Открытие папки. File → Open Folder. Создаём проект.
- 40–50 мин: Первый файл. Новый файл → index.html. Пишем базовую разметку.
- 50–55 мин: Сохранение. Ctrl+S. Автосохранение (File → Auto Save).
- 55–60 мин: Итоги. Бадж «Пульт Активирован».

### 🛠️ Практика

Ребёнок скачивает и устанавливает VS Code. Создаёт папку `pirate-project`. Открывает её как проект. Создаёт файл `index.html`. Пишет:

```html
<!DOCTYPE html>
<html>
<head><title>Пиратский Штаб</title></head>
<body><h1>Бип! Termix на связи!</h1></body>
</html>
```

Сохраняет. Открывает в браузере (правый клик → Reveal in File Explorer → открыть в браузере).

🎮 Игровая механика

Игра «Найди элемент». Наставник говорит: «Найди строку состояния внизу. Что там написано?» Дети ищут и отвечают. Кто первый — жетон.

🏠 Миссия

· 🥉 Юнга: Установить VS Code, создать папку, создать файл README.md.
· 🥈 Боцман: Создать index.html с заголовком и абзацем.
· 🥇 Штурман: Создать три файла (.html, .css, .js) в одной папке.

📖 ЗАНЯТИЕ 2: ТЕРМИНАЛ — КОМАНДНЫЙ МОСТИК

🎯 Цель

Освоить встроенный терминал VS Code и базовые команды.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Терминал — это командный мостик! Отсюда ты отдаёшь приказы компьютеру. Не мышкой — словом!»
· 5–15 мин: Открытие терминала. Terminal → New Terminal. Горячая клавиша.
· 15–25 мин: Базовые команды. pwd (где я), ls (что тут), cd (перейти), mkdir (создать папку).
· 25–40 мин: Создание файлов через терминал. touch index.html. echo "Hello" > file.txt.
· 40–50 мин: Навигация. Переходы между папками, возврат (cd ..).
· 50–55 мин: Очистка. clear. Итоги. Бадж «Командир Терминала».

🛠️ Практика

Ребёнок открывает терминал в VS Code (Ctrl+`). Выполняет:

```bash
pwd                    # посмотреть, где мы
ls                     # посмотреть, что в папке
mkdir pirate-storage   # создать папку
cd pirate-storage      # перейти в неё
touch treasure.txt     # создать файл
echo "золото" >> treasure.txt  # записать в файл
cat treasure.txt       # прочитать файл
cd ..                  # вернуться обратно
clear                  # очистить экран
```

🎮 Игровая механика

Квест «Построй крепость». Наставник диктует: «Создай папку fort. Зайди в неё. Создай файлы wall.txt, gate.txt, tower.txt.» Дети выполняют. Кто первый построил — жетон.

🏠 Миссия

· 🥉 Юнга: Создать папку и файл через терминал.
· 🥈 Боцман: Создать структуру из 3 папок с файлами.
· 🥇 Штурман: Написать в терминале историю из 5 команд с пояснениями.

📖 ЗАНЯТИЕ 3: РАСШИРЕНИЯ — УСИЛИТЕЛИ

🎯 Цель

Научиться устанавливать расширения и использовать их.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «VS Code — как я без рыбы. Хорош, но можно лучше! Расширения — это супер-силы!»
· 5–15 мин: Что такое расширения. Marketplace. Как искать и устанавливать.
· 15–25 мин: Темы. One Dark Pro, Dracula, GitHub Theme. Меняем внешний вид.
· 25–40 мин: Prettier. Форматирование кода одним нажатием.
· 40–50 мин: Live Server. Запуск сайта в браузере с автообновлением.
· 50–55 мин: Итоги. Бадж «Усилитель».

🛠️ Практика

Ребёнок открывает Extensions (Ctrl+Shift+X). Устанавливает:

· One Dark Pro — тема (меняем цвет редактора).
· Prettier — форматирование (настраиваем Format on Save).
· Live Server — запуск сайта (правый клик → Open with Live Server).

Проверяем: открываем index.html, жмём Go Live. Браузер открывается. Меняем HTML, сохраняем — страница обновляется автоматически.

🎮 Игровая механика

Конкурс «Чей редактор красивее?». Дети выбирают тему, меняют шрифт, показывают друг другу. Голосование стикерами.

🏠 Миссия

· 🥉 Юнга: Установить тему и поменять цвет редактора.
· 🥈 Боцман: Установить Prettier и настроить автоформатирование.
· 🥇 Штурман: Установить все три расширения и написать инструкцию для друга.

📖 ЗАНЯТИЕ 4: ХОТКЕИ — СКОРОСТЬ КАПИТАНА

🎯 Цель

Освоить горячие клавиши для ускорения работы.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Мышь — это якорь. Хоткеи — это паруса! Сегодня учимся летать по коду!»
· 5–15 мин: Главные хоткеи. Ctrl+S, Ctrl+Z, Ctrl+C, Ctrl+V, Ctrl+X.
· 15–25 мин: Навигация. Ctrl+P (файлы), Ctrl+Shift+P (команды), Ctrl+B (боковая панель).
· 25–40 мин: Редактирование. Ctrl+D (выделить такое же), Alt+↑↓ (двигать строку), Ctrl+/ (комментарий).
· 40–50 мин: Поиск и замена. Ctrl+F (поиск), Ctrl+H (замена).
· 50–55 мин: Итоги. Бадж «Скоростной Капитан».

🛠️ Практика

Ребёнок открывает файл с кодом (подготовленный наставником). Выполняет задания:

· Найти слово «Termix» через Ctrl+F.
· Заменить «Termix» на «Капитан Termix» через Ctrl+H.
· Закомментировать строку (Ctrl+/).
· Продублировать строку (Shift+Alt+↓).
· Переместить строку вверх (Alt+↑).
· Открыть другой файл (Ctrl+P → ввести имя).
· Спрятать боковую панель (Ctrl+B).

🎮 Игровая механика

Соревнование «Хоткей-батл». Наставник говорит действие. Дети выполняют без мыши. Кто первый — жетон.

Примеры: «Сохрани файл», «Найди слово error», «Замени все fish на gold», «Перемести строку вниз».

🏠 Миссия

· 🥉 Юнга: Выучить 5 хоткеев (сохранить, копировать, вставить, отменить, найти).
· 🥈 Боцман: Выучить 10 хоткеев и записать их в файл шпаргалкой.
· 🥇 Штурман: Пройти «Хоткей-батл» у наставника на скорость.

📖 ЗАНЯТИЕ 5: СНИППЕТЫ — ЗАКЛИНАНИЯ

🎯 Цель

Освоить Emmet и собственные сниппеты.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Писать каждый тег руками? Скучно! Сниппеты пишут код за тебя. Это как заклинания!»
· 5–15 мин: Что такое Emmet. ! + Tab — готовый HTML. div*3 — три дива.
· 15–25 мин: CSS через Emmet. m10 → margin: 10px. bgc → background-color.
· 25–40 мин: Свои сниппеты. File → Preferences → Configure Snippets.
· 40–50 мин: Создаём сниппет termix → вставляет приветствие Termix'а.
· 50–55 мин: Итоги. Бадж «Заклинатель Сниппетов».

🛠️ Практика

Ребёнок открывает HTML-файл. Пишет ! → жмёт Tab. Получает готовый шаблон.

Пробует Emmet:

· div.container>ul>li*5 → вложенная структура.
· p*3>lorem → три абзаца с текстом.
· a[href="#"]{Ссылка} → ссылка с атрибутом и текстом.

CSS:

· w100 → width: 100px.
· c#fff → color: #fff.
· fz20 → font-size: 20px.

Свой сниппет: termix → вставляет:

```
<!-- 🐱 Termix Academy -->
<h1>Бип! Привет, юнга!</h1>
```

🎮 Игровая механика

Соревнование «Сниппет-гонка». Наставник говорит: «Сделай список из 5 пунктов с ссылками». Дети пишут через Emmet. Кто быстрее — жетон.

🏠 Миссия

· 🥉 Юнга: Сделать страницу через ! и Emmet.
· 🥈 Боцман: Создать 3 своих сниппета.
· 🥇 Штурман: Сниппет, который вставляет шаблон пиратской страницы (header, main, footer).
