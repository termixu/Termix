## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: C# → синтаксис → «похож на Java» → «зачем ещё один язык?» → скучно.

Подход Termix Academy: C# → арсенал Unity → каждый скрипт — это поведение → «я программирую на том же языке, на котором делают Hollow Knight и Cuphead!».

C# — это язык от Microsoft, на котором пишут игры на Unity, приложения для Windows, веб-сервисы и многое другое. Termix учит детей использовать C# как основной инструмент для создания игр и приложений. Строгая типизация, LINQ, свойства, события — всё это становится понятным через практику.

```csharp
using System.Linq;

var inventory = new List<string> { "Золотой дублон", "Рыба-меч", "Карта", "Ром" };

Console.WriteLine("Инвентарь Termix'а:");
inventory.ForEach(item => Console.WriteLine($"- {item}"));

var longItems = inventory.Where(i => i.Length > 4).ToList();
Console.WriteLine($"\nДлинные названия: {string.Join(", ", longItems)}");

var sorted = inventory.OrderBy(i => i).ToList();
Console.WriteLine($"\nПо алфавиту: {string.Join(", ", sorted)}");
```

🛠️ Практика

```csharp
class Pirate
{
    public string Name { get; set; }
    public string Rank { get; set; }
    public int Gold { get; private set; }
    
    public Pirate(string name, string rank, int gold)
    {
        Name = name;
        Rank = rank;
        Gold = gold;
    }
    
    public void AddGold(int amount)
    {
        Gold += amount;
        Console.WriteLine($"{Name} получил {amount} дублонов. Теперь у него {Gold}.");
    }
    
    public override string ToString() => $"{Rank} {Name} — {Gold} дублонов";
}

var termix = new Pirate("Termix", "Капитан", 999);
Console.WriteLine(termix);
termix.AddGold(50);
```
