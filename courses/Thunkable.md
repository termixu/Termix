---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: мобильная разработка → Swift/Kotlin → сложно → «я не смогу сделать приложение».

Подход Termix Academy: Thunkable → визуальные блоки → собрал как пазл → запустил на телефоне → «моё приложение работает и на iOS, и на Android!».

Thunkable — это среда для создания мобильных приложений без написания кода. Всё в браузере. Компоненты перетаскиваются на экран телефона. Логика собирается из визуальных блоков. В отличие от App Inventor, Thunkable работает и на iOS. Termix учит детей создавать кроссплатформенные приложения: от простой «Нажималки» до мини-игр с камерой и облачными данными.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Понимать структуру мобильного приложения.
- Работать в визуальной среде Thunkable.
- Добавлять компоненты: кнопки, текст, изображения, камеру.
- Программировать логику через визуальные блоки.
- Использовать переменные и условия.
- Работать с экранами (многостраничные приложения).
- Сохранять данные в облаке.
- Тестировать приложение на реальном телефоне.
- Публиковать приложение.

## 📖 ЗАНЯТИЕ 1: ПЕРВАЯ КНОПКА

### 🎯 Цель

Освоить интерфейс Thunkable, создать первое приложение с кнопкой.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Бип! Добро пожал onboard карманного корабля 2.0! Thunkable — это верфь, где рождаются приложения для iOS и Android. Погнали!»
- 5–15 мин: Вход через thunkable.com. Интерфейс: Design и Blocks.
- 15–25 мин: Режим Design. Добавление компонентов: Button, Label.
- 25–40 мин: Режим Blocks. Собираем блок: When Button.Click → set Label.Text to «Бип!».
- 40–50 мин: Тестирование на телефоне (приложение Thunkable Live) или веб-превью.
- 50–55 мин: Сохранение проекта. Итоги. Бадж «Первая кнопка».

### 🛠️ Практика

Ребёнок открывает Thunkable. Создаёт новый проект «PirateApp». В Design перетаскивает Button (текст «Нажми меня!») и Label (пустой). Переходит в Blocks. Собирает: when Button1.Click → set Label1.Text to «Бип! Привет, юнга!». Запускает через Thunkable Live. Нажимает кнопку — видит текст.

### 🎮 Игровая механика

Игра «Кто быстрее?». Наставник: «Добавь вторую кнопку, которая меняет цвет фона экрана». Кто первый — жетон.

### 🏠 Миссия

## 📖 ЗАНЯТИЕ 2: ПИРАТСКИЙ КАЛЬКУЛЯТОР

### 🎯 Цель

Освоить переменные и ввод данных.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Считаем золото! Переменные — это сундучки для чисел. Вводим данные, считаем, выводим результат!»
- 5–15 мин: Компонент TextInput (ввод текста). Переменные в Blocks.
- 15–30 мин: Сложение. Вводим два числа, нажимаем кнопку — результат.
- 30–45 мин: Все 4 действия: +, -, *, /. Выбор действия через выпадающий список.
- 45–55 мин: Форматирование вывода. «Ответ: 42 золотых дублонов».
- 55–60 мин: Итоги. Бадж «Счетовод».

### 🛠️ Практика

Ребёнок добавляет два TextInput (Число 1, Число 2), Button «Сложить», Label (результат). В Blocks:

- Initialize app variable result to 0.
- When Button.Click → set app variable result to TextInput1.Text + TextInput2.Text.
- Set Label.Text to join «Результат: » get app variable result.

Для 4 действий добавляет ListPicker с вариантами «+», «-», «*», «/» и условие if-else.

### 🎮 Игровая механика

Соревнование «Калькулятор». Наставник даёт 5 примеров. Дети считают в своём приложении. Кто быстрее — жетон.

### 🏠 Миссия

- 🥉 Юнга: Сложение двух чисел.
- 🥈 Боцман: Все 4 действия.
- 🥇 Штурман: Добавить проверку: если деление на ноль — вывести ошибку.

## 📖 ЗАНЯТИЕ 4: МИНИ-ИГРА — КЛИКЕР

### 🎯 Цель

Освоить счётчики и таймеры.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Кликер! Нажимай на меня как можно быстрее. Таймер тикает. Счёт растёт. Играем!»
- 5–15 мин: Счётчик кликов. Переменная score.
- 15–30 мин: Таймер. Countdown 10 секунд.
- 30–45 мин: Игра «Кликер». За 10 секунд накликать максимум.
- 45–55 мин: Рекорд. Сохранение лучшего результата.
- 55–60 мин: Итоги. Бадж «Кликер».

### 🛠️ Практика

Ребёнок добавляет Button (большая, по центру), Label (счёт), Label (таймер). Переменные: score, timeLeft = 10. При клике на кнопку: score += 1, обновить Label. Timer: каждую секунду timeLeft -= 1. Когда timeLeft = 0 → завершить игру, показать результат.

### 🎮 Игровая механика

Турнир «Кликер». Дети играют в кликеры друг друга. Топ-3 — жетоны.

### 🏠 Миссия

- 🥉 Юнга: Счётчик кликов.
- 🥈 Боцман: Добавить таймер на 10 секунд.
- 🥇 Штурман: Сохранять рекорд между запусками.
