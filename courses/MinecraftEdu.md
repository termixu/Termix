---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: Minecraft → просто игра → «я строю домик» → никакого программирования.

Подход Termix Academy: Minecraft Education → Agent-помощник → код на Python или блоках → «я запрограммировал робота, и он построил пирамиду!».

Minecraft Education Edition — это образовательная версия Minecraft с встроенным Code Builder. Дети программируют Agent'а (маленького робота), который выполняет команды: строит стены, копает шахты, сажает деревья. Termix ведёт юных шахтёров от первых блоков до сложных программ с циклами и условиями.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Программировать в среде Minecraft Education.
- Использовать Code Builder (блочный или Python).
- Управлять Agent'ом: двигать, строить, разрушать.
- Применять циклы и условия в программировании.
- Решать пространственные задачи.
- Работать в команде (совместные постройки).
- Понимать алгоритмы и последовательности.

## 📖 ЗАНЯТИЕ 1: AGENT — ПЕРВЫЙ ПОМОЩНИК

### 🎯 Цель

Установить Minecraft Education, открыть Code Builder, написать первую программу.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Бип! Добро пожаловать в шахты знаний! Minecraft Education — это не просто игра. Здесь живёт Agent — твой робот-помощник. Погнали программировать!»
- 5–15 мин: Установка Minecraft Education (education.minecraft.net). Вход с учебной учётной записью.
- 15–25 мин: Знакомство с Code Builder (клавиша C). Блочный редактор и Python.
- 25–40 мин: Первая программа. Agent, move forward by 1.
- 40–50 мин: Agent двигается на 5 шагов вперёд, поворачивает, возвращается.
- 50–55 мин: Сохранение мира. Итоги. Бадж «Agent-активатор».

### 🛠️ Практика

Ребёнок открывает Minecraft Education. Создаёт мир (шаблон «Стартовый мир»). Нажимает C → открывается Code Builder. Выбирает блочный редактор. Собирает программу: Agent move forward by 3. Нажимает «Запустить». Agent идёт!

### 🎮 Игровая механика

Игра «Кто дальше?». Наставник: «Запрограммируй Agent'а пройти 10 шагов, повернуть направо и пройти ещё 5. Кто точнее — жетон».

### 🏠 Миссия

- 🥉 Юнга: Agent идёт 3 шага вперёд.
- 🥈 Боцман: Agent идёт по квадрату (вперёд, поворот, вперёд, поворот, вперёд, поворот, вперёд).
- 🥇 Штурман: Agent идёт по треугольнику.

## 📖 ЗАНЯТИЕ 2: СТРОИМ БАШНЮ

### 🎯 Цель

Освоить циклы (repeat) для повторяющихся действий.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Башня! Высокая, до облаков. Но строить каждый этаж вручную — скучно. Цикл "повторить" сделает всё сам!»
- 5–15 мин: Цикл repeat (повторить N раз).
- 15–30 мин: Agent строит башню: move up, place block, repeat.
- 30–45 мин: Башня из 10 этажей одним циклом.
- 45–55 мин: Раскрашивание этажей. Каждый этаж — новый цвет.
- 55–60 мин: Итоги. Бадж «Строитель башен».

### 🛠️ Практика

Ребёнок программирует Agent'а:
- Repeat 10 times:
  - Agent move forward by 1
  - Agent move up by 1
  - Agent place block (камень)

Agent строит лестницу-башню. Для раскрашивания добавляет условие: если этаж чётный — камень, нечётный — доски.

### 🎮 Игровая механика

Соревнование «Кто выше?». Дети строят башни. У кого выше за 5 минут — жетон.

### 🏠 Миссия

- 🥉 Юнга: Построить башню из 5 этажей.
- 🥈 Боцман: Построить башню из 10 этажей с разными цветами.
- 🥇 Штурман: Построить башню-спираль (Agent поворачивает на каждом этаже).

## 📖 ЗАНЯТИЕ 3: ПИРАТСКИЙ СУНДУК

### 🎯 Цель

Освоить условия (if) и взаимодействие с блоками.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Сундук с золотом! Но не все блоки — золото. Agent должен проверять: если блок жёлтый — собрать. Если нет — идти дальше!»
- 5–15 мин: Условие if. Проверка блока.
- 15–30 мин: Agent идёт по линии блоков и проверяет каждый.
- 30–45 мин: Программа: если блок = золотой → собрать (destroy block), иначе → пропустить.
- 45–55 мин: Подсчёт собранного золота.
- 55–60 мин: Итоги. Бадж «Золотоискатель».

### 🛠️ Практика

Ребёнок расставляет линию из 10 блоков: 5 золотых, 5 каменных. Программирует Agent'а:
- Repeat 10 times:
  - If block in front is gold → Agent destroy block, count +1
  - Agent move forward by 1

### 🎮 Игровая механика

Соревнование «Кто больше соберёт?». Наставник расставляет блоки в случайном порядке. Дети программируют Agent'а.

### 🏠 Миссия

- 🥉 Юнга: Agent проверяет 5 блоков.
- 🥈 Боцман: Agent проверяет 10 блоков и считает золото.
- 🥇 Штурман: Agent обходит поле 3x3 и собирает всё золото.

## 📖 ЗАНЯТИЕ 4: ЛАБИРИНТ

### 🎯 Цель

Освоить пространственное мышление и алгоритмы поиска пути.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Лабиринт! Agent должен найти выход. Налево? Направо? Прямо? Программируем разведчика!»
- 5–15 мин: Планирование маршрута. Схема лабиринта на бумаге.
- 15–30 мин: Программирование Agent'а для прохождения лабиринта.
- 30–45 мин: Тестирование и отладка.
- 45–55 мин: Прохождение лабиринта за минимальное число шагов.
- 55–60 мин: Итоги. Бадж «Разведчик».

### 🛠️ Практика

Ребёнок строит лабиринт из стен. Планирует маршрут. Программирует Agent'а:
- Move forward
- Turn right
- Move forward
- Turn left
- ...

Agent проходит лабиринт.

### 🎮 Игровая механика

Соревнование «Кто быстрее пройдёт?». Одинаковый лабиринт. У кого программа короче — жетон.

### 🏠 Миссия

- 🥉 Юнга: Пройти простой лабиринт (3 поворота).
- 🥈 Боцман: Пройти лабиринт с тупиком.
- 🥇 Штурман: Пройти лабиринт за минимальное число шагов.

## 📖 ЗАНЯТИЕ 5: МОСТ ЧЕРЕЗ ПРОПАСТЬ

### 🎯 Цель

Освоить планирование и строительство сложных структур.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Пропасть! Agent должен построить мост. Шаг за шагом, блок за блоком. Инженерное дело!»
- 5–15 мин: Планирование моста. Длина, материал, опоры.
- 15–30 мин: Программирование Agent'а: move forward, place block below.
- 30–45 мин: Строительство моста через пропасть.
- 45–55 мин: Украшение моста. Факелы, перила.
- 55–60 мин: Итоги. Бадж «Мостостроитель».

### 🛠️ Практика

Ребёнок находит пропасть (или строит её). Программирует Agent'а:
- Repeat длина моста:
  - Agent move forward by 1
  - Agent place block below (камень)

Agent строит мост, шагая над пропастью.

### 🎮 Игровая механика

Конкурс «Самый длинный мост». Кто построит длиннее за 5 минут — жетон.

### 🏠 Миссия

- 🥉 Юнга: Построить мост длиной 5 блоков.
- 🥈 Боцман: Построить мост длиной 10 блоков с факелами.
- 🥇 Штурман: Построить арочный мост.

## 📖 ЗАНЯТИЕ 6: ФЕРМА

### 🎯 Цель

Освоить автоматизацию повторяющихся задач.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Ферма! Agent сажает семена, собирает урожай. Всё автоматически. Как робот-фермер!»
- 5–15 мин: Планирование фермы. Грядки, вода, семена.
- 15–30 мин: Программирование: Agent move, place block (семена).
- 30–45 мин: Автоматический сбор урожая: Agent destroy block (пшеница).
- 45–55 мин: Расширение фермы. Цикл для нескольких грядок.
- 55–60 мин: Итоги. Бадж «Фермер».

### 🛠️ Практика

Ребёнок строит грядку 5x1. Программирует Agent'а:
- Посадка: repeat 5 (move forward, place block — семена).
- Сбор: repeat 5 (move forward, destroy block — пшеница).

### 🎮 Игровая механика

Соревнование «Чья ферма больше?». Дети расширяют фермы. У кого больше грядок обработано — жетон.

### 🏠 Миссия

- 🥉 Юнга: Посадить 5 семян.
- 🥈 Боцман: Посадить и собрать урожай.
- 🥇 Штурман: Сделать ферму 3x3 с автоматической посадкой и сбором.
