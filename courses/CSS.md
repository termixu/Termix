## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: CSS → селекторы → свойства → «почему оно не центрируется?!» → «я ненавижу вёрстку».

Подход Termix Academy: CSS → палитра капитана → покрасил палубу → добавил тени → «мой сайт выглядит как настоящий!».

CSS — это язык стилей, который превращает скучный HTML в красивый сайт. Цвета, шрифты, отступы, анимации — всё это CSS. Termix учит детей не бояться каскада и делать сайты, на которые приятно смотреть. К концу курса ребёнок оформит многостраничный сайт Termix Academy в пиратском стиле.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Подключать CSS к HTML (внешние, внутренние, inline).
- Использовать селекторы: теги, классы, id.
- Работать с цветами и фонами.
- Управлять шрифтами и текстом.
- Понимать блочную модель (margin, border, padding, content).
- Использовать Flexbox для выравнивания.
- Позиционировать элементы.
- Создавать анимации и переходы.
- Использовать CSS-переменные.

## 📖 ЗАНЯТИЕ 1: КИСТЬ И ПАЛИТРА

### 🎯 Цель

Подключить CSS к HTML, освоить селекторы и цвета.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Бип! HTML — это чёрно-белый чертёж. CSS — это краски! Берём кисть и красим корабль!»
- 5–15 мин: Три способа подключения CSS: inline, внутренний (`<style>`), внешний (`<link>`).
- 15–30 мин: Синтаксис. Селекторы по тегу, классу (`.`) и id (`#`).
- 30–50 мин: Цвета: `color`, `background-color`. Именованные, HEX, RGB.
- 50–55 мин: Приоритет селекторов. Кто главнее?
- 55–60 мин: Итоги. Бадж «Кисть».

### 🛠️ Практика

Ребёнок создаёт `style.css` и подключает к `index.html`:

**index.html:**
```html
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Termix Academy</h1>
    <p class="pirate-text">Бип! Добро пожаловать!</p>
    <p id="special">Секретное сообщение</p>
</body>
</html>
```

style.css:

```css
h1 {
    color: #00ff41;
    background-color: #0a0a0a;
}

.pirate-text {
    color: #ffd700;
}

#special {
    color: #ff0040;
}
```

Игра «Раскрась по приказу». Наставник: «Сделай все заголовки h2 зелёными, а параграфы — жёлтыми». Дети пишут селекторы.

📖 ЗАНЯТИЕ 2: ШРИФТЫ И ФЛАГИ

🎯 Цель

Освоить управление шрифтами и текстом.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Флаги на мачтах! Шрифт — это твой флаг. Выбери правильный, и тебя узнают издалека!»
· 5–15 мин: font-family. Веб-безопасные шрифты. Google Fonts.
· 15–30 мин: font-size, font-weight, font-style.
· 30–50 мин: text-align, text-decoration, text-transform, line-height.
· 50–55 мин: Подключение Google Fonts.
· 55–60 мин: Итоги. Бадж «Флаг».

🛠️ Практика

```css
@import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap');

h1 {
    font-family: 'Press Start 2P', monospace;
    font-size: 32px;
    text-align: center;
    text-transform: uppercase;
}

p {
    font-size: 18px;
    line-height: 1.5;
}

.gold {
    color: #ffd700;
    font-weight: bold;
    text-decoration: underline;
}
```

📖 ЗАНЯТИЕ 3: БЛОЧНАЯ МОДЕЛЬ — ЯЩИКИ

🎯 Цель

Освоить блочную модель CSS.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Ящики! Каждый элемент — это сундук. У него есть стенки (border), прокладка (padding) и отступ (margin)!»
· 5–15 мин: Блочная модель: content → padding → border → margin.
· 15–30 мин: width, height. max-width, min-height.
· 30–50 мин: border, border-radius. padding, margin.
· 50–55 мин: box-sizing: border-box. Зачем оно нужно.
· 55–60 мин: Итоги. Бадж «Ящик».

🛠️ Практика

```css
.card {
    width: 300px;
    padding: 20px;
    margin: 10px auto;
    border: 2px solid #00ff41;
    border-radius: 12px;
    background-color: #0a0a0a;
    color: #ffffff;
    box-sizing: border-box;
}

.card h3 {
    margin-top: 0;
}

.card p {
    margin-bottom: 0;
}
```

```html
<div class="card">
    <h3>Termix</h3>
    <p>Капитан. Золото: 999.</p>
</div>
```

📖 ЗАНЯТИЕ 4: ФОНЫ И ТЕНИ

🎯 Цель

Освоить фоны, градиенты и тени.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Тени! Даже у корабля есть тень. А у сайта — box-shadow. Добавим глубины!»
· 5–15 мин: background-color, background-image, background-size.
· 15–30 мин: Градиенты: linear-gradient, radial-gradient.
· 30–50 мин: Тени: box-shadow, text-shadow.
· 50–55 мин: Многослойные фоны.
· 55–60 мин: Итоги. Бадж «Тень».

🛠️ Практика

```css
body {
    background: linear-gradient(135deg, #0a0a0a, #1a2f1a);
    color: #ffffff;
}

.card {
    background: linear-gradient(180deg, #1a1a1a, #0d0d0d);
    box-shadow: 0 4px 15px rgba(0, 255, 65, 0.3);
    text-shadow: 0 0 10px rgba(0, 255, 65, 0.5);

.glow-text {
    color: #00ff41;
    text-shadow: 0 0 5px #00ff41, 0 0 10px #00ff41, 0 0 20px #00ff41;
```

📖 ЗАНЯТИЕ 5: FLEXBOX — ВЫРАВНИВАНИЕ

🎯 Цель

Освоить Flexbox.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Flexbox! Это магия, которая расставляет всё по местам. Матросы — в ряд! Пушки — по центру!»
· 5–15 мин: display: flex. Главная и поперечная оси.
· 15–30 мин: justify-content, align-items, align-content.
· 30–50 мин: flex-direction, flex-wrap, gap.
· 50–55 мин: Собираем навигацию и галерею карточек.
· 55–60 мин: Итоги. Бадж «Flex-капитан».

🛠️ Практика

```css
nav {
    display: flex;
    justify-content: center;
    gap: 20px;
    background-color: #0a0a0a;
    padding: 15px;
}

nav a {
    color: #00ff41;
    text-decoration: none;
}

.gallery {
    display: flex;
    flex-wrap: wrap;
    justify-content: center;
    gap: 20px;
}

.card {
    flex: 1 1 250px;
    max-width: 300px;
}
```

🎮 Игровая механика

Соревнование «Расставь экипаж». Наставник: «Сделай 5 карточек в ряд через Flexbox. При сужении экрана карточки переносятся». Кто быстрее — жетон.

🏠 Миссия

· 🥉 Юнга: Сделать горизонтальную навигацию через Flexbox.
· 🥈 Боцман: Сделать галерею карточек.

📖 ЗАНЯТИЕ 6: ПОЗИЦИОНИРОВАНИЕ — КАРТА

🎯 Цель

Освоить позиционирование элементов.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Карта! Каждый элемент знает свои координаты. Можно закрепить флаг на мачте или спрятать сундук в углу!»
· 5–15 мин: position: static (по умолчанию).
· 15–30 мин: relative, absolute, fixed, sticky.
· 30–50 мин: top, right, bottom, left. z-index.
· 50–55 мин: Практика. Фиксированная шапка, абсолютный значок, sticky-меню.
· 55–60 мин: Итоги. Бадж «Картограф».

🛠️ Практика

```css
header {
    position: fixed;
    top: 0;
    left: 0;
    right: 0;
    background: #0a0a0a;
    padding: 15px;
    z-index: 100;
}

.card {
    position: relative;
}

.badge {
    position: absolute;
    top: -10px;
    right: -10px;
    background: #ff0040;
    color: white;
    border-radius: 50%;
    width: 24px;
    height: 24px;
    text-align: center;
}
```

📖 ЗАНЯТИЕ 7: АНИМАЦИЯ — ОЖИВЛЕНИЕ

🎯 Цель

Освоить переходы и анимации.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Оживление! Кнопка должна реагировать на нажатие. Карточка — выезжать. Сайт — дышать!»
· 5–15 мин: transition. Плавные изменения.
· 15–30 мин: transform: translate, rotate, scale, skew.
· 30–50 мин: @keyframes и animation.
· 50–55 мин: Практика. Анимированная кнопка, пульсирующий значок.
· 55–60 мин: Итоги. Бадж «Аниматор».

🛠️ Практика

```css
.button {
    background: #00ff41;
    color: #0a0a0a;
    padding: 12px 24px;
    border: none;
    border-radius: 8px;
    cursor: pointer;
    transition: transform 0.2s, box-shadow 0.2s;
}

.button:hover {
    transform: scale(1.05);
    box-shadow: 0 0 20px rgba(0, 255, 65, 0.5);
}

@keyframes pulse {
    0%, 100% { opacity: 1; }
    50% { opacity: 0.5; }
}

.termix-tail {
    animation: pulse 2s ease-in-out infinite;
}

@keyframes slideIn {
    from { transform: translateX(-100%); opacity: 0; }
    to { transform: translateX(0); opacity: 1; }
}

.card {
    animation: slideIn 0.5s ease-out;
}
```

🎮 Игровая механика

Конкурс «Самая смешная анимация». Дети анимируют Termix'а. Голосование стикерами.

🏠 Миссия
