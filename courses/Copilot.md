---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: Copilot → «он пишет за меня» → я не понимаю код → «я разучился программировать».

Подход Termix Academy: Copilot → «он предлагает, я проверяю» → я понимаю каждую строчку → «я стал быстрее и умнее».

GitHub Copilot — это AI-ассистент, который живёт прямо в редакторе кода. Он дописывает строки, предлагает функции, объясняет код. Termix учит детей использовать Copilot не как костыль, а как ускоритель: писать спецификации, читать подсказки, проверять сгенерированный код и улучшать его.

🛠️ Практика

Ребёнок пишет спецификацию:

```python
# Функция: count_treasure
# Вход: список строк (названия предметов)
# Выход: словарь (предмет → количество)
# Пример:
#   вход: ["золото", "рыба", "золото"]
#   выход: {"золото": 2, "рыба": 1}
```

Copilot генерирует:

```python
def count_treasure(items):
    result = {}
    for item in items:
        if item in result:
            result[item] += 1
        else:
            result[item] = 1
    return result
```

Сравнивает с версией от Copilot без спецификации (просто «# посчитай предметы»).

```

def sum_positive(numbers: list[int]) -> int:
    """Возвращает сумму всех положительных чисел в списке."""
    return sum(num for num in numbers if num > 0)
```

🛠️ Практика

Ребёнок пишет функцию:

```python
def divide_treasure(gold: int, pirates: int) -> int:
    if pirates == 0:
        raise ValueError("Пиратов не может быть ноль!")
    return gold // pirates
```

Ниже пишет комментарий для Copilot:

```python
# Напиши тесты для divide_treasure
# Проверь: обычное деление, деление на 1, деление на 0 (ошибка)
```

Copilot генерирует:

```python
import pytest

def test_divide_treasure_normal():
    assert divide_treasure(100, 4) == 25

def test_divide_treasure_by_one():
    assert divide_treasure(100, 1) == 100

def test_divide_treasure_by_zero():
    with pytest.raises(ValueError):
        divide_treasure(100, 0)
```

Запускает: pytest test_treasure.py -v. Видит зелёные галочки.

Соревнование «Найди edge-кейс». Наставник даёт функцию. Дети просят Copilot сгенерировать тесты. У кого больше edge-кейсов покрыто — жетон.
