---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: JavaScript → синтаксис → переменные → страшно → «я ещё маленький».

Подход Termix Academy: JS Robot → игрушечный робот → напиши ему команду → он покрасил клетку → «Я УПРАВЛЯЮ РОБОТОМ!».

JS Robot (или аналогичная среда вроде «Роботландии») — это образовательная среда для самых первых шагов в текстовом программировании. Ребёнок управляет виртуальным роботом, который закрашивает клетки на поле, обходит препятствия и доходит до цели. Вместо визуальных блоков — настоящие команды на упрощённом языке, очень похожем на JavaScript.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Понимать, что такое текстовое программирование.
- Писать простые команды: move, turn, paint.
- Использовать циклы (for, while) для повторения действий.
- Использовать условия (if) для проверки клеток.
- Планировать маршрут на клетчатом поле.
- Находить и исправлять ошибки в коде.


📖 ЗАНЯТИЕ 3: ЦИКЛЫ — ПОВТОРЯЙ!

🎯 Цель

Освоить цикл for.

⏱️ Тайминг (45–60 минут)

· 0–5 мин: Termix: «Повторять одно и то же 10 раз? Скучно! Цикл сделает это за тебя!»
· 5–10 мин: Цикл for (let i = 0; i < 5; i++).
· 10–20 мин: Закрашиваем полосу из 10 клеток.
· 20–30 мин: Рисуем лесенку (цикл + поворот внутри).
· 30–40 мин: Рисуем квадрат одним циклом.
· 40–50 мин: Практика. Робот строит стену.
· 50–55 мин: Итоги. Бадж «Циклопират».

🛠️ Практика

```javascript
// Полоса из 10 клеток
set_color("green");
for (let i = 0; i < 10; i++) {
    paint();
    move();
}

// Лесенка
for (let i = 0; i < 5; i++) {
    paint();
    move();
    turn_right();
    paint();
    move();
    turn_left();
}
```

📖 ЗАНЯТИЕ 4: ЛАБИРИНТ С УСЛОВИЯМИ

🎯 Цель

Освоить условие if.

⏱️ Тайминг (45–60 минут)

· 0–5 мин: Termix: «Лабиринт! Не всегда ясно, куда идти. Условие if проверит дорогу!»
· 5–10 мин: Условие if. Проверка через is_free_ahead().
· 10–20 мин: Робот идёт, если впереди свободно.
· 20–30 мин: Робот выбирает путь: если впереди стена — повернуть.
· 30–40 мин: Лабиринт с выбором на каждом шагу.
· 40–50 мин: Практика. Пройти лабиринт с тупиками.
· 50–55 мин: Итоги. Бадж «Логик».

🛠️ Практика

```javascript
for (let i = 0; i < 20; i++) {
    if (is_free_ahead()) {
        move();
        paint();
    } else {
        turn_right();
    }
}
```

🎮 Игровая механика

Игра «Слепой лабиринт». Наставник меняет лабиринт. Программа с if должна работать на любом раскладе.

📖 ЗАНЯТИЕ 5: ЦИКЛЫ WHILE

🎯 Цель

Освоить цикл while.

⏱️ Тайминг (45–60 минут)

· 0–5 мин: Termix: «While! Пока впереди свободно — иди. Сам остановится у стены!»
· 5–10 мин: Цикл while (is_free_ahead()).
· 10–20 мин: Робот идёт до стены.
· 20–30 мин: Робот закрашивает коридор до упора.
· 30–40 мин: Комбинация while + if.
· 40–50 мин: Практика. Робот обходит комнату по периметру.
· 50–55 мин: Итоги. Бадж «Упорный пират».

🛠️ Практика

```javascript
// Идти до стены и красить
while (is_free_ahead()) {
    paint();
    move();
}

// Обход комнаты (стена справа)
while (is_free_ahead()) {
    move();
    paint();
    if (is_free_right()) {
        turn_right();
    }
}
```

🎮 Игровая механика

Соревнование «Универсальная программа». Наставник меняет размер коридора. Программа должна работать всегда.

🏠 Миссия

· 🥉 Юнга: while до стены.
· 🥈 Боцман: Закрасить коридор.
· 🥇 Штурман: Обойти комнату по периметру.

📖 ЗАНЯТИЕ 7: ФУНКЦИИ-ПОМОЩНИКИ

🎯 Цель

Освоить создание своих функций.

⏱️ Тайминг (45–60 минут)

· 0–5 мин: Termix: «Функции! Помощники робота. Один раз написал — много раз вызвал!»
· 5–10 мин: Создание функции. function.
· 10–20 мин: Функция go_and_paint().
· 20–30 мин: Функция turn_around().
· 30–40 мин: Функция draw_square(size).
· 40–50 мин: Практика. Строим город функций.
· 50–55 мин: Итоги. Бадж «Командир».

🛠️ Практика

```javascript
function go_and_paint(steps) {
    for (let i = 0; i < steps; i++) {
        paint();
        move();
    }
}

function turn_around() {
    turn_left();
    turn_left();
}

go_and_paint(5);
turn_around();
go_and_paint(5);
```

📖 ЗАНЯТИЕ 8: БОЛЬШОЙ ЛАБИРИНТ

· 50–55 мин: Итоги. Бадж «Лабиринтщик».

🛠️ Практика

```javascript
function follow_wall() {
    while (is_free_ahead()) {
        move();
        paint();
        if (is_free_right()) {
            turn_right();
            move();
            turn_right();
        }
    }
    turn_left();
}

for (let i = 0; i < 4; i++) {
    follow_wall();
}
```
