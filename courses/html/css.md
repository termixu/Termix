## 📖 ТЕОРИЯ (15 МИНУТ)

### Что такое CSS

CSS — это язык стилей. Он не добавляет элементы на страницу. Он говорит, как уже существующие элементы должны выглядеть.

Три кита, на которых стоит CSS:
1. **Селектор** — КОГО красим? (заголовок, абзац, кнопку).
2. **Свойство** — ЧТО меняем? (цвет, размер, отступ).
3. **Значение** — НА ЧТО меняем? (зелёный, 32 пикселя, по центру).

### Три способа добавить CSS на страницу

**1. Inline (внутри тега).** Быстро, но грязно:
```html
<h1 style="color: green;">Заголовок</h1>
```

2. Внутренний (в <style>). Для одной страницы:

```html
<head>
    <style>
        h1 { color: green; }
    </style>
</head>
```

3. Внешний (в отдельном файле). Самый правильный способ:

```html
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

Внешний файл — это как пиратский устав. Один раз написал правила — и они работают на всех страницах.

Как выглядит CSS-правило

```css
селектор {
    свойство: значение;
    свойство: значение;
}
```

Пример:

```css
h1 {
    color: #00ff41;
    text-align: center;
    font-size: 48px;
}
```

Это значит: «Все заголовки <h1> сделать зелёными, по центру и размером 48 пикселей».

Пиратская аналогия

· Селектор = «Эй, боцман!» (обращаемся к конкретному человеку).
· Свойство = «Перекрась пушки» (что делаем).
· Значение = «В золотой цвет» (как делаем).

🛠️ ПРАКТИКА (30 МИНУТ)

Шаг 1: Создаём CSS-файл

Ребёнок создаёт файл style.css в той же папке, где index.html.

Шаг 2: Подключаем CSS к HTML

В index.html, внутри <head>, добавляем:

```html
<link rel="stylesheet" href="style.css">
```

Шаг 3: Пишем первые стили

В style.css:

```css
body {
    background-color: #0a0a0a;
    color: #ffffff;
    font-family: Arial, sans-serif;
}

h1 {
    color: #00ff41;
    text-align: center;
    font-size: 48px;
}

p {
    color: #ffd700;
    font-size: 18px;
}
```

Сохраняем. Обновляем страницу в браузере. Видим:

· Чёрный фон.
· Зелёный заголовок по центру.
· Золотые абзацы.

Шаг 4: Экспериментируем

Меняем цвета на свои. Меняем размер шрифта. Добавляем text-align: right — текст уезжает вправо.

Игра «Что главнее?»

Наставник подключает к одному HTML два CSS: в одном h1 { color: red; }, в другом h1 { color: blue; }. Дети угадывают, какой цвет применится. Объяснение: «Кто последний — того и цвет. Как в пиратском споре: кто громче крикнул — того и добыча!»
