## 📖 ТЕОРИЯ (15 МИНУТ)

### Что такое HTML

HTML расшифровывается как **HyperText Markup Language** — «язык гипертекстовой разметки». Звучит страшно, но на деле это просто **команды**, которые говорят браузеру: «Здесь будет заголовок», «Здесь — картинка», «Здесь — кнопка».

Каждая команда называется **тегом**. Теги похожи на пиратские приказы. Они всегда в треугольных скобках: `<приказ>`. И у большинства приказов есть начало и конец: `<приказ>` и `</приказ>`.

### Три главных тега, с которых начинается любой сайт

- `<html>` — «Это HTML-документ». Всё, что внутри — код страницы.
- `<head>` — «Голова». Здесь служебная информация: заголовок страницы, настройки.
- `<body>` — «Тело». Всё, что мы видим на странице: текст, картинки, кнопки.

### Как это выглядит

```html
<!DOCTYPE html>
<html>
<head>
    <title>Пиратский Штаб</title>
</head>
<body>
    Бип! Привет, юнга!
</body>
</html>
```

· <title> — заголовок вкладки браузера. То, что написано сверху.
· Внутри <body> — любой текст, и он появится на странице.

Почему HTML — это каркас

Если сайт — это пиратский корабль:

· <html> — сам корабль. Всё внутри него.
· <head> — капитанский журнал (не видно, но важно).
· <body> — палуба и трюм (всё, что видно).
· <title> — флаг на мачте (видно издалека, на вкладке).

Без HTML нет сайта. CSS и JS могут отсутствовать — сайт будет некрасивым и неподвижным, но он будет. А без HTML — пустота.

🛠️ ПРАКТИКА (30 МИНУТ)

Создаём первую страницу

Ребёнок открывает VS Code. Создаёт файл index.html. Пишет:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пиратский Штаб</title>
</head>
<body>
    <h1>Бип! Добро пожаловать на борт!</h1>
    <p>Меня зовут Termix. Я — кибер-пиратский кот.</p>
    <p>Это мой сайт. Он пока что просто каркас. Но скоро мы его раскрасим!</p>
</body>
</html>
```

Открывает файл в браузере. Видит заголовок и два абзаца.

Эксперимент: что будет, если убрать HTML?

Наставник открывает DevTools (F12) → Elements. Показывает, что у любого сайта есть <html>, <head>, <body>. Удаляет их (временно, не сохраняя) — страница ломается. Возвращает обратно. Дети видят: HTML — это фундамент.

Игра «Найди тег»

Наставник открывает любой сайт (например, Wikipedia). Дети в DevTools ищут теги <html>, <head>, <body>, <title>. Кто первый нашёл все четыре — жетон.
