## 📖 ТЕОРИЯ (15 МИНУТ)

### Что такое JavaScript

JavaScript — это язык программирования, который работает прямо в браузере. Он умеет:
- Реагировать на клики и нажатия клавиш.
- Менять текст и стили на лету.
- Считать, проверять, запоминать.
- Общаться с сервером (загружать данные без перезагрузки страницы).

### Где живёт JavaScript

Есть три способа добавить JavaScript на страницу:

**1. Внутри HTML (в теге `<script>`):**
```html
<script>
    alert("Бип! Привет, юнга!");
</script>
```

2. Внешний файл (правильный способ):

```html
<script src="script.js"></script>
```

3. В консоли браузера (для экспериментов):
F12 → Console → пишем alert("Бип!") → Enter.

Первая команда: alert

Самая простая команда JavaScript:

```javascript
alert("Бип! Привет, юнга!");
```

Что происходит:

1. Браузер читает команду.
2. Понимает: «Нужно показать всплывающее окно».
3. Показывает окно с текстом «Бип! Привет, юнга!».
4. Пользователь нажимает OK.

Вторая команда: console.log

```javascript
console.log("Это сообщение видно только в консоли");
```

Эта команда тише. Она не пугает пользователя всплывающими окнами. Она пишет сообщение в консоль (F12 → Console). Программисты используют её для отладки.

Пиратская аналогия

· HTML = корабль (каркас).
· CSS = флаг и окраска (стиль).
· JavaScript = капитан, который орёт команды («Поднять паруса!», «Огонь из пушек!»).
· alert = рында (громкий сигнал всем).
· console.log = запись в бортовом журнале (тихо, только для капитана).

## 🛠️ ПРАКТИКА (30 МИНУТ)

Шаг 1: Создаём JavaScript-файл

Ребёнок создаёт файл script.js в той же папке, где index.html и style.css.

Шаг 2: Подключаем JavaScript к HTML

В index.html, перед закрывающим тегом </body>, добавляем:

```html
<script src="script.js"></script>
```

Почему в конце body? Чтобы HTML загрузился раньше, чем скрипты начнут выполняться.

Шаг 3: Пишем первый код

В script.js:

```javascript
alert("Бип! Привет, юнга! Добро пожаловать на борт!");
console.log("Страница загружена. Хвост зелёный.");
```

Сохраняем. Обновляем страницу. Видим всплывающее окно. Открываем консоль — видим сообщение.

Шаг 4: Кнопка, которая что-то делает

Добавляем в index.html кнопку:

```html
<button id="fireButton">Огонь из пушек!</button>
<p id="message"></p>
```

В script.js пишем:

```javascript
// Находим кнопку на странице по id
let button = document.querySelector("#fireButton");

// Находим параграф для сообщения
let message = document.querySelector("#message");

// Говорим: "Когда на кнопку нажмут — сделай это"
button.addEventListener("click", function() {
    message.textContent = "БА-БАХ! Попадание в цель!";
    console.log("Пушки выстрелили!");
});
```

Обновляем страницу. Нажимаем кнопку. Появляется текст!

Шаг 5: Экспериментируем

Меняем текст сообщения. Добавляем подсчёт выстрелов:

```javascript
let shots = 0;

button.addEventListener("click", function() {
    shots = shots + 1;
    message.textContent = "БА-БАХ! Выстрелов: " + shots;
});
```

Каждый клик — счётчик растёт. Дети видят: страница ожила.
