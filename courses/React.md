## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: React → компоненты → хуки → «слишком сложно, вернусь в HTML».

Подход Termix Academy: React → капитанский мостик → каждый компонент — прибор → «я собрал своё приложение из кубиков!».

React — это библиотека JavaScript для создания пользовательских интерфейсов. Её используют Facebook, Instagram, Netflix и миллионы разработчиков. Termix учит детей мыслить компонентами: разбивать интерфейс на части, управлять состоянием, передавать данные через пропсы. К концу курса ребёнок соберёт своё приложение Termix Academy на React.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Понимать, что такое компонентный подход.
- Создавать React-приложение с Vite.
- Писать JSX и понимать его отличие от HTML.
- Использовать компоненты и пропсы.
- Управлять состоянием через useState.
- Обрабатывать события в React.
- Использовать useEffect для побочных эффектов.
- Работать со списками и ключами.
- Подключать API и отображать данные.
- Деплоить React-приложение на Vercel.

## 📖 ЗАНЯТИЕ 1: ПЕРВАЯ РУБКА

### 🎯 Цель

Понять, что такое React, создать проект через Vite, написать первый JSX.

### ⏱️ Тайминг (90 минут)

- 0–5 мин: Termix: «Бип! Добро пожаловать на капитанский мостик! React — это твоя рубка управления интерфейсом. Погнали!»
- 5–15 мин: Что такое React. Компоненты. Почему Facebook его создал.
- 15–30 мин: Vite. Создание проекта: `npm create vite@latest`.
- 30–50 мин: Структура проекта. `src/App.jsx`. Первый JSX.
- 50–70 мин: Меняем контент. Добавляем заголовок, параграф, картинку Termix'а.
- 70–80 мин: Горячая перезагрузка. Меняем код — видим результат мгновенно.
- 80–90 мин: Итоги. Бадж «Первая Рубка».

### 🛠️ Практика

Ребёнок создаёт проект:

```bash
npm create vite@latest termix-bridge -- --template react
cd termix-bridge
npm install
npm run dev
