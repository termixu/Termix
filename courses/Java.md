```java
class Pirate {
    private String name;
    private String rank;
    private int gold;
    
    public Pirate(String name, String rank, int gold) {
        this.name = name;
        this.rank = rank;
        this.gold = gold;
    }
    
    public void introduce() {
        System.out.println("Бип! Я " + rank + " " + name + ". У меня " + gold + " дублонов.");
    }
    
    public void addGold(int amount) {
        gold += amount;
        System.out.println(name + " получил " + amount + " дублонов. Теперь у него " + gold + ".");
    }
}

public class Main {
    public static void main(String[] args) {
        Pirate termix = new Pirate("Termix", "Капитан", 999);
        Pirate vasya = new Pirate("Вася", "Юнга", 10);
        
        termix.introduce();
        vasya.introduce();
        vasya.addGold(20);
    }
}
```

⏱️ Тайминг (90 минут)

· 0–5 мин: Termix: «Шторм! Корабль может перевернуться. Исключения ловят ошибки и не дают программе утонуть!»
· 5–15 мин: Что такое исключения. try-catch.

· 45–60 мин: Проверяемые и непроверяемые исключения.

· 75–90 мин: Итоги. Бадж «Штурмовик».

🛠️ Практика

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        try {
            System.out.print("Введи делимое: ");
            int a = Integer.parseInt(scanner.nextLine());
            
            System.out.print("Введи делитель: ");
            int b = Integer.parseInt(scanner.nextLine());
            
            int result = a / b;
            System.out.println("Результат: " + result);
            
        } catch (ArithmeticException e) {
            System.out.println("Ошибка! На ноль делить нельзя!");
        } catch (NumberFormatException e) {
            System.out.println("Ошибка! Это не число!");
        } finally {
            System.out.println("Бип! Операция завершена.");
        }
    }
}
```

🛠️ Практика

```java
import java.io.*;

public class Main {
    public static void main(String[] args) {
        // Запись
        try (BufferedWriter writer = new BufferedWriter(new FileWriter("logbook.txt", true))) {
            writer.write("День 1: Нашёл золотой дублон.");
            writer.newLine();
            writer.write("День 2: Шторм. Всё мокрое.");
            writer.newLine();
        } catch (IOException e) {
            System.out.println("Ошибка записи: " + e.getMessage());
        }
        
        // Чтение
        try (BufferedReader reader = new BufferedReader(new FileReader("logbook.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            System.out.println("Ошибка чтения: " + e.getMessage());
        }
    }
}
```
