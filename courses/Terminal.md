> **Версия:** 1.0
>
> **Возраст:** 10–14 лет
>
> **Длительность:** 8 занятий по 60–75 минут
>
> **Проводник:** Termix — кибер-пиратский кот, Хранитель Ядра
>
> **Принцип:** Не бояться чёрного экрана, а видеть в нём пульт управления. Не зубрить команды, а играть в пиратский квест.

---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: терминал → чёрный экран → мигающий курсор → «я боюсь сломать компьютер».

Подход Termix Academy: терминал → капитанский мостик → `ls`, `cd`, `mkdir` → «я управляю компьютером голосом!».

Терминал — это самый быстрый способ общаться с компьютером. Без мыши. Без иконок. Только текст. Termix учит детей не бояться командной строки, а видеть в ней инструмент силы: создавать папки и файлы, перемещаться по системе за доли секунды, управлять файлами и понимать, как устроена операционная система изнутри. Всё в формате пиратского квеста: каждая команда — это заклинание, каждая папка — остров, а оболочка — это сам корабль.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Открывать терминал и не бояться мигающего курсора.
- Понимать, где он находится в файловой системе (`pwd`, `ls`, `cd`).
- Создавать, копировать, перемещать и удалять файлы и папки (`mkdir`, `touch`, `cp`, `mv`, `rm`).
- Просматривать содержимое файлов (`cat`, `head`, `tail`, `less`).
- Использовать автодополнение (Tab) и историю команд (стрелки вверх/вниз).
- Понимать флаги команд (`-l`, `-a`, `-r`, `-f`, `--help`).
- Использовать пайпы (`|`) и перенаправление ввода-вывода (`>`, `>>`, `<`).
- Писать простые скрипты автоматизации на Bash (петли, переменные).
- Настраивать свою оболочку (`.bashrc`, алиасы, приглашение командной строки).

## 📖 ЗАНЯТИЕ 1: ПЕРВАЯ КОМАНДА

### 🎯 Цель

Открыть терминал, понять, где мы находимся и что вокруг.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Бип! Добро пожаловать на Командный мостик! Этот чёрный экран — твой штурвал. С него ты управляешь всем кораблём. Погнали!»
- 5–10 мин: Как открыть терминал. Windows (PowerShell / WSL / Git Bash), Mac (Terminal), Linux (Ctrl+Alt+T).
- 10–20 мин: `pwd` (Print Working Directory) — команда, которая показывает, в какой ты папке. Твой компас.
- 20–35 мин: `ls` (List) — что тут? Список файлов и папок в текущей директории.
- 35–45 мин: `cd` (Change Directory) — путешествие по папкам. `cd Папка`, `cd ..` (назад), `cd ~` (домой), `cd /` (корень).
- 45–55 мин: Практика. Путешествие по файловой системе. Найти папку `Downloads` и вернуться домой.
- 55–60 мин: Итоги. Бадж «Штурман».

### 🛠️ Практика

```bash
pwd                    # где я? (например, /home/юнга)
ls                     # что здесь? (Documents, Downloads, Desktop...)
cd Desktop             # перейти на Рабочий стол
pwd                    # теперь я на /home/юнга/Desktop
ls                     # посмотреть файлы на рабочем столе
cd ..                  # вернуться назад (в /home/юнга)
cd ~                   # перейти в домашнюю папку (быстрый путь)
cd /                   # перейти в корень системы (опасно, но посмотреть можно)
ls                     # посмотреть, что в корне (bin, etc, home, usr...)
cd ~                   # быстро домой
```

🎮 Игровая механика

Игра «Навигатор». Наставник говорит: «Перейди в папку Documents, потом вернись, потом перейди в Downloads». Дети выполняют. Кто первый — жетон.

🏠 Миссия

· 🥉 Юнга: Выполнить pwd, ls, cd Desktop, cd .., cd ~.
· 🥈 Боцман: Обойти 5 разных папок и вернуться домой.
· 🥇 Штурман: Зайти в корень (cd /), посмотреть что там (ls), найти папку etc и вернуться домой.

📖 ЗАНЯТИЕ 2: СТРОИМ КРЕПОСТЬ

🎯 Цель

Научиться создавать папки и файлы.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Пирату нужна крепость! Сегодня строим базу: папки-комнаты, файлы-сокровища!»
· 5–15 мин: mkdir (Make Directory) — создать папку. Можно создать несколько за раз: mkdir fort walls tower.
· 15–25 мин: touch — создать пустой файл. touch README.md. Если файл уже есть, touch обновит его дату изменения, но содержимое не тронет.
· 25–35 мин: echo — вывести текст или записать его в файл. echo "золото" > treasure.txt.
· 35–50 мин: Строим крепость. Папка fort, внутри walls.txt, gate.txt.
· 50–55 мин: Итоги. Бадж «Строитель».

🛠️ Практика

```bash
mkdir pirate-fort                # создать папку
cd pirate-fort                   # зайти в неё
mkdir walls tower gate           # создать три папки сразу
ls                               # проверить: walls, tower, gate
touch walls/stone.txt            # создать пустой файл в папке walls
echo "крепкая стена" > walls/stone.txt   # записать текст в файл (перезапишет, если там что-то было)
cat walls/stone.txt              # прочитать, что записали
touch treasure.txt               # создать файл в корне крепости
echo "золото" >> treasure.txt    # дописать текст в конец файла (без перезаписи)
cat treasure.txt                 # проверить
```

🎮 Игровая механика

Квест «Построй крепость». Наставник диктует структуру: «Создай папку fort, внутри rooms, в rooms — файл chest.txt с текстом gold». Дети выполняют. Кто первый — жетон.

🏠 Миссия

· 🥉 Юнга: Создать папку и 3 файла в ней.
· 🥈 Боцман: Создать структуру из 3 папок с файлами внутри. Записать в файлы текст через echo.
· 🥇 Штурман: Создать «пиратскую базу»: 5 папок, 10 файлов, в каждом — описание сокровища.

📖 ЗАНЯТИЕ 3: КОПИРУЕМ СОКРОВИЩА

🎯 Цель

Освоить копирование, перемещение и удаление файлов.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Сокровища нужно прятать! Копировать в секретные папки, перемещать в сундуки, удалять улики!»
· 5–15 мин: cp — копировать. cp файл папка/. cp -r папка1/ папка2/ для копирования директорий.
· 15–25 мин: mv — переместить или переименовать. mv старый_файл новый_файл.
· 25–35 мин: rm — удалить. ОСТОРОЖНО! Нет корзины. rm файл, rm -r папка/ (рекурсивно).
· 35–50 мин: Практика. Квест «Спрячь сокровище».
· 50–55 мин: Итоги. Бадж «Хранитель Сокровищ».

🛠️ Практика

```bash
cp treasure.txt treasure-backup.txt   # копировать файл (создать копию)
cp -r tower/ tower-backup/            # копировать папку (рекурсивно)
ls                                     # проверить, что копии создались
mv treasure.txt secret-vault/         # переместить оригинал в папку
mv old-name.txt new-name.txt          # переименовать файл
rm old-file.txt                        # удалить файл (безвозвратно)
rm -r old-folder/                      # удалить папку с содержимым
```

🎮 Игровая механика

Квест «Спрячь сокровище». Наставник: «Создай папку secret, скопируй туда treasure.txt, переименуй в hidden.txt, удали оригинал». Дети выполняют цепочку.

🏠 Миссия

· 🥉 Юнга: Скопировать файл и переименовать копию.
· 🥈 Боцман: Переместить файл в другую папку и удалить старый.
· 🥇 Штурман: Пройти квест из 5 действий без ошибок.

📖 ЗАНЯТИЕ 4: ЧИТАЕМ КАРТЫ

🎯 Цель

Освоить команды для просмотра файлов.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Карта сокровищ — это файл. Нужно уметь его читать! Не только открывать целиком, но и заглядывать в начало и конец!»
· 5–15 мин: cat — вывести содержимое файла целиком. Хорошо для маленьких файлов.
· 15–25 мин: head и tail — показать первые или последние 10 строк. head -n 5 file покажет 5 строк.
· 25–40 мин: less — умный просмотр больших файлов. Листать стрелками. /term — искать слово "term". q — выход.
· 40–50 мин: wc — посчитать строки, слова и символы. wc -l file — только строки.
· 50–55 мин: Итоги. Бадж «Картограф».

🛠️ Практика

```bash
cat diary.txt              # вывести весь файл (если он маленький)
head -n 3 diary.txt        # первые 3 строки
tail -n 3 diary.txt        # последние 3 строки
wc -l diary.txt            # посчитать, сколько строк в файле
less long-story.txt        # листать длинный файл (q — выход)
                           # Находясь в less: /pirate — найти слово "pirate", n — следующее совпадение
```

🎮 Игровая механика

Игра «Найди секрет». Наставник создаёт файл из 20 строк, в одной из которых спрятано кодовое слово. Дети должны найти его через grep или просмотр в less. Кто первый — жетон.

🏠 Миссия

· 🥉 Юнга: Прочитать файл через cat.
· 🥈 Боцман: Найти первую и последнюю строку файла через head и tail.
· 🥇 Штурман: Посчитать количество строк в файле и найти секретное слово через grep.

📖 ЗАНЯТИЕ 5: ФЛАГИ И ПАЙПЫ

🎯 Цель

Освоить продвинутое использование команд: флаги, пайплайны и перенаправление.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Флаги — это режимы. Как у моего хвоста: зелёный, жёлтый, красный. А пайпы (|) — это канаты, связывающие команды в цепочку!»
· 5–15 мин: Флаги: -l (подробный список), -a (показать скрытые файлы), -r (рекурсивно), -f (форсировать).
· 15–25 мин: grep — поиск текста. grep "золото" *.txt ищет "золото" во всех текстовых файлах.
· 25–40 мин: Пайпы (|) — передать вывод одной команды на вход другой. ls -la | grep ".txt".
· 40–50 мин: Перенаправление: > (перезаписать файл), >> (добавить в конец), < (читать из файла).
· 50–55 мин: Итоги. Бадж «Флагман».

🛠️ Практика

```bash
ls -la                    # подробный список со скрытыми файлами
ls -la | grep ".md"      # найти все Markdown-файлы в выводе ls
grep "золото" *.txt      # найти слово "золото" во всех txt-файлах
cat diary.txt | grep "шторм"  # показать только строки со словом "шторм"
echo "новая строка" >> diary.txt  # добавить строку в конец файла
sort diary.txt > sorted_diary.txt # отсортировать строки и сохранить в новый файл
```

🎮 Игровая механика

Игра «Расшифруй флаг». Наставник пишет команду с флагами. Дети угадывают, что она сделает. Кто первый — жетон.

🏠 Миссия

· 🥉 Юнга: Использовать ls -l и grep.
· 🥈 Боцман: Создать цепочку из двух команд через |.
· 🥇 Штурман: Найти все файлы, изменённые в этом месяце, и отсортировать их по размеру.

📖 ЗАНЯТИЕ 6: ОБОЛОЧКА — НАШ КОРАБЛЬ

🎯 Цель

Понять, что такое оболочка (Bash), и настроить её под себя.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Терминал — это не просто окно. Им управляет оболочка — Bash. Это наш корабль. Настроим его под себя!»
· 5–15 мин: Что такое Bash, Zsh, Fish. Как узнать свою оболочку: echo $SHELL.
· 15–25 мин: echo $PATH — где оболочка ищет программы.
· 25–40 мин: Алиасы (alias). alias ll='ls -la'. Сохранение в .bashrc.
· 40–50 мин: Настройка приглашения командной строки (PS1).
· 50–55 мин: Итоги. Бадж «Капитан Оболочки».

🛠️ Практика

```bash
echo $SHELL                # узнать свою оболочку
echo $PATH                 # посмотреть пути поиска программ
alias ll='ls -la'          # создать временный алиас
ll                         # использовать алиас
nano ~/.bashrc             # открыть файл настроек Bash
# Добавить строку: alias pirateship='echo "Бип! Корабль к плаванию готов!"'
source ~/.bashrc           # применить изменения без перезапуска терминала
pirateship                 # использовать новый алиас
```

🎮 Игровая механика

Соревнование «Чей терминал красивее?». Дети настраивают приглашение командной строки (PS1) — добавляют цвет, эмодзи, имя. Самый красивый — жетон.

🏠 Миссия

· 🥉 Юнга: Создать алиас ll для ls -la.
· 🥈 Боцман: Добавить 3 полезных алиаса в .bashrc.
· 🥇 Штурман: Настроить цветное приглашение командной строки с именем пирата.

📖 ЗАНЯТИЕ 7: АВТОМАТИЗАЦИЯ — МАШИННОЕ ОТДЕЛЕНИЕ

🎯 Цель

Освоить написание bash-скриптов.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Автоматизация! Чтобы не повторять команды вручную, мы пишем скрипты. Один раз написал — запускаешь вечно!»
· 5–15 мин: Что такое скрипт. Shebang (#!/bin/bash). chmod +x script.sh.
· 15–30 мин: Переменные в Bash. name="Termix", echo "Привет, $name!".
· 30–45 мин: Циклы и условия. for, while, if.
· 45–55 мин: Аргументы скрипта ($1, $2, $@).
· 55–60 мин: Итоги. Бадж «Автоматизатор».

🛠️ Практика

```bash
nano greet.sh             # создать скрипт
```

```bash
#!/bin/bash
echo "Бип! Привет, капитан!"
echo "Сегодня: $(date)"
echo "Ты находишься в: $(pwd)"
echo "Файлов вокруг: $(ls | wc -l)"
```

```bash
chmod +x greet.sh         # сделать скрипт исполняемым
./greet.sh                # запустить скрипт
```

🎮 Игровая механика

Конкурс «Самый полезный скрипт». Дети придумывают и пишут скрипт. Номинации: самый полезный, самый смешной.

📖 ЗАНЯТИЕ 8: БОЛЬШОЙ КВЕСТ

🎯 Цель

Применить все изученные команды в комплексном приключении.

⏱️ Тайминг (60–75 минут)

· 0–5 мин: Termix: «Сегодня — Большой Квест! Ты получишь карту и должен найти сокровище, используя ВСЕ команды!»
· 5–10 мин: Выдача карты (список инструкций).
· 10–45 мин: Выполнение квеста.
· 45–55 мин: Проверка. У кого сокровище найдено?
· 55–60 мин: Награждение. Баджи, жетоны, сертификаты.

🛠️ Практика

Сценарий квеста:

1. Создай папку island.
2. Внутри создай папки beach, forest, cave.
3. В beach создай файл message.txt с текстом: «Ищи в пещере».
4. Скопируй message.txt в cave/.
5. В cave создай папку hidden.
6. Перемести message.txt в hidden/.
7. Переименуй message.txt в treasure.txt.
8. Допиши в treasure.txt текст: «Сокровище найдено!».
9. Прочитай treasure.txt и покажи наставнику.
