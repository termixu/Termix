# 💻 КУРС «TERMINAL: КОМАНДНЫЙ МОСТИК»

> **Версия:** 1.0
>
> **Возраст:** 10–14 лет
>
> **Длительность:** 6 занятий по 60 минут
>
> **Проводник:** Termix — кибер-пиратский кот
>
> **Принцип:** Не бояться чёрного экрана. Не зубрить команды, а играть в пиратский квест.

---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: терминал → чёрный экран → страшно → «я сломаю компьютер».

Подход Termix Academy: терминал → капитанский мостик → `ls`, `cd`, `mkdir` → «я управляю компьютером голосом!».

Терминал — это самый быстрый способ общаться с компьютером. Без мыши. Без иконок. Только текст. Termix учит детей не бояться командной строки, а видеть в ней инструмент силы: создавать папки, файлы, перемещаться по системе за доли секунды. Всё в формате пиратского квеста: каждая команда — это заклинание, каждая папка — остров.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Открывать терминал и не бояться.
- Понимать, где он находится (`pwd`).
- Смотреть содержимое папок (`ls`).
- Перемещаться по файловой системе (`cd`).
- Создавать папки и файлы (`mkdir`, `touch`).
- Копировать, перемещать, удалять (`cp`, `mv`, `rm`).
- Читать содержимое файлов (`cat`, `less`).
- Понимать флаги команд (`-l`, `-a`, `-r`, `-f`).
- Использовать автодополнение (Tab).

---

## 🗺️ КАРТА КУРСА

- **Занятие 1 — «Первая команда».** Открытие терминала. `pwd`, `ls`, `cd`. Где я и что вокруг.
- **Занятие 2 — «Строим крепость».** `mkdir`, `touch`, `echo`. Создание папок и файлов.
- **Занятие 3 — «Копируем сокровища».** `cp`, `mv`, `rm`. Копирование, перемещение, удаление.
- **Занятие 4 — «Читаем карты».** `cat`, `less`, `head`, `tail`. Чтение файлов.
- **Занятие 5 — «Флаги: Усилители».** `-l`, `-a`, `-r`, `-f`, `--help`. Продвинутое использование.
- **Занятие 6 — «Большой квест».** Комплексное приключение. Построй базу, найди сокровище.

## 📖 ЗАНЯТИЕ 1: ПЕРВАЯ КОМАНДА

### 🎯 Цель

Открыть терминал, понять где мы находимся и что вокруг.

### ⏱️ Тайминг (60 минут)

- 0–5 мин: Termix: «Бип! Добро пожаловать на командный мостик! Этот чёрный экран — твой штурвал. Погнали!»
- 5–10 мин: Как открыть терминал. Windows (Terminal/PowerShell), Mac (Terminal), Linux (Ctrl+Alt+T).
- 10–20 мин: `pwd` — где я? Команда показывает текущую папку.
- 20–30 мин: `ls` — что тут? Список файлов и папок.
- 30–45 мин: `cd` — перемещение. `cd Папка`, `cd ..` (назад), `cd ~` (домой).
- 45–55 мин: Практика. Путешествие по папкам.
- 55–60 мин: Итоги. Бадж «Штурман».

### 🛠️ Практика

```bash
pwd                    # где я?
ls                     # что здесь?
cd Desktop            # перейти на рабочий стол
pwd                    # проверить
ls                     # посмотреть файлы на рабочем столе
cd ..                  # вернуться назад
cd ~                   # перейти в домашнюю папку
