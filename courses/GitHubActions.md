## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: GitHub Actions → YAML → workflow → сложно → «я просто хочу чтобы оно само работало».

Подход Termix Academy: GitHub Actions → автоматическая канонада → запушил код → пушки выстрелили → сайт обновился → «магия!».

GitHub Actions — это встроенный в GitHub инструмент автоматизации. Каждый раз, когда ты пушишь код, Actions может запустить тесты, собрать проект, выложить сайт на хостинг или отправить уведомление в Telegram. Termix учит детей настраивать CI/CD (Continuous Integration / Continuous Deployment) — то, чем занимаются настоящие DevOps-инженеры.

· 55–60 мин: Итоги. Бадж «Запальщик».

🛠️ Практика

Ребёнок создаёт workflow .github/workflows/triggers.yml:

```yaml
name: Арсенал Триггеров
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]
  schedule:
    - cron: '0 9 * * 1'  # каждый понедельник в 9 утра
  workflow_dispatch:       # ручной запуск
jobs:
  fire:
    runs-on: ubuntu-latest
    steps:
      - name: Выстрел
        run: echo "Пушка сработала от триггера: ${{ github.event_name }}"
```

Ручной запуск: Actions → выбираем workflow → Run workflow.

📖 ЗАНЯТИЕ 4: ТЕСТЫ — ПРИСТРЕЛКА

🎯 Цель

Настроить автоматический запуск тестов при пуше.

⏱️ Тайминг (75–90 минут)

· 0–5 мин: Termix: «Перед выстрелом нужно пристреляться! Тесты — это проверка, что ядро летит куда надо!»
· 5–15 мин: Что такое тесты. Зачем их автоматизировать.
· 15–30 мин: Пишем простой тест на Python.
· 30–50 мин: Workflow, который запускает тесты.
· 50–55 мин: Симуляция ошибки. Ломаем тест — смотрим красный крестик.
· 55–60 мин: Итоги. Бадж «Наводчик».

🛠️ Практика

Ребёнок создаёт файл test_pirate.py:

```python
def test_gold():
    gold = 100
    assert gold > 0, "Золото должно быть больше нуля!"

def test_termix():
    name = "Termix"
    assert name == "Termix", "Это не Термикс!"
```

Workflow .github/workflows/test.yml:

```yaml
name: Тесты
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      - name: Установка pytest
        run: pip install pytest
      - name: Запуск тестов
        run: pytest test_pirate.py
```

Зелёная галочка — тесты прошли. Ломаем тест: assert gold > 200 → красный крестик.

🛠️ Практика

Ребёнок создаёт простой сайт в папке site/:

site/index.html:

```html
<h1>Бип! Termix Academy!</h1>
<p>Этот сайт задеплоен через Actions!</p>
```

Workflow .github/workflows/deploy.yml:

```yaml
name: Деплой
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: actions/checkout@v4
      - name: Деплой на GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./site
```

После пуша сайт доступен по адресу https://username.github.io/repo/.

📖 ЗАНЯТИЕ 6: КАНОНАДА

🎯 Цель

Собрать комплексный проект: тесты + деплой + уведомление.

⏱️ Тайминг (75–90 минут)

· 0–5 мин: Termix: «Канонада! Все пушки разом! Тесты. Деплой. Уведомление! Полный боевой цикл!»
· 5–10 мин: План канонады. Что должно происходить при пуше.
· 10–50 мин: Сборка комплексного workflow.
· 50–65 мин: Финальный пуш. Смотрим на результат.
· 65–75 мин: Награждение. Баджи, жетоны, сертификаты.

🛠️ Практика

Workflow .github/workflows/cannonade.yml:

```yaml
name: Канонада
on:
  push:
    branches: [main]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      - run: pip install pytest
      - run: pytest test_pirate.py

  deploy:
    needs: test
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: actions/checkout@v4
      - uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./site

  notify:
    needs: deploy
    runs-on: ubuntu-latest
    steps:
      - name: Уведомление
        run: echo "ВСЕ ПУШКИ ОТРАБОТАЛИ! Сайт обновлён!"
```
