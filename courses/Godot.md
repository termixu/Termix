## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: Godot → сцены → ноды → сигналы → «слишком сложно, вернусь в Scratch».

Подход Termix Academy: Godot → игровая верфь → корабль плывёт → Termix прыгает → «я сделал платформер!».

Godot — это бесплатный игровой движок с открытым исходным кодом. На нём делают инди-игры, которые выходят в Steam. Termix учит детей собирать игры из сцен и нодов: персонажи, физика, анимации, UI. К концу курса ребёнок соберёт платформер с котом-пиратом и сможет выложить его на itch.io.

- 50–70 мин: Движение спрайта в 4 направлениях.
- 70–80 мин: Ограничение движения границами экрана.
- 80–90 мин: Итоги. Бадж «Штурвальный».

### 🛠️ Практика

Ребёнок создаёт скрипт для Termix'а:

```gdscript
extends Sprite2D

var speed = 300

func _process(delta):
    var direction = Vector2.ZERO
    if Input.is_action_pressed("ui_right"):
        direction.x += 1
    if Input.is_action_pressed("ui_left"):
        direction.x -= 1
    if Input.is_action_pressed("ui_down"):
        direction.y += 1
    if Input.is_action_pressed("ui_up"):
        direction.y -= 1
    
    position += direction.normalized() * speed * delta
```

🛠️ Практика

```gdscript
extends CharacterBody2D

var speed = 300
var jump_velocity = -400
var gravity = 800

func _physics_process(delta):
    velocity.y += gravity * delta
    
    var direction = Input.get_axis("ui_left", "ui_right")
    velocity.x = direction * speed
    
    if Input.is_action_just_pressed("ui_accept") and is_on_floor():
        velocity.y = jump_velocity
    
    move_and_slide()
```
