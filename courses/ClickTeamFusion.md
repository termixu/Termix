> **Принцип:** Не писать код, а собирать игры из событий. Не учить синтаксис, а видеть результат сразу.

---

## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: Clickteam Fusion → странный интерфейс → «я не понимаю, как здесь программировать».

Подход Termix Academy: Clickteam Fusion → фабрика игр → Termix прыгает → враги двигаются → «я сделал платформер без единой строчки кода!».

Clickteam Fusion — это среда для создания игр без программирования. Вместо кода — таблица событий: WHEN (что случилось) → THEN (что сделать). Никакого синтаксиса. Никаких точек с запятой. Termix учит детей создавать настоящие игры: платформеры, стрелялки, аркады.

---

## 🎯 ЧЕМУ НАУЧИТСЯ РЕБЁНОК

- Понимать событийно-ориентированную логику (event-driven).
- Создавать игры без написания кода.
- Работать с графикой, звуком и анимациями.
- Настраивать физику: гравитацию, столкновения.
- Создавать врагов с простым AI.
- Проектировать уровни и экраны.
- Экспортировать готовую игру.

## 📖 ЗАНЯТИЕ 1: ПЕРВАЯ КОМНАТА

### 🎯 Цель

Освоить интерфейс Clickteam Fusion, создать первый объект и первое событие.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Бип! Добро пожаловать на фабрику игр! Clickteam Fusion — это место, где игры рождаются без кода. Погнали!»
- 5–15 мин: Интерфейс. Frame Editor, Event Editor, панель объектов.
- 15–30 мин: Создание фрейма (комнаты). Размер, фон.
- 30–45 мин: Создание объекта Termix'а. Инструменты рисования или импорт спрайта.
- 45–55 мин: Первое событие. When (нажата стрелка) → Then (двигаться вправо).
- 55–60 мин: Запуск игры. Итоги. Бадж «Первая комната».

### 🛠️ Практика

Ребёнок открывает Clickteam Fusion. Создаёт новый проект. В Frame Editor рисует фон (зелёный остров). Создаёт объект Active (рисует оранжевый круг — Termix). В Event Editor добавляет событие: Upon pressing "Right Arrow" → Set X position to X + 5. Запускает. Termix двигается!

### 🎮 Игровая механика

Игра «Кто первый до края». Наставник: «Заставь Termix'а дойти до правого края экрана». У кого Termix быстрее — жетон (меняем скорость).

### 🏠 Миссия

- 🥉 Юнга: Создать объект и двигать вправо.
- 🥈 Боцман: Движение во все 4 стороны.
- 🥇 Штурман: Termix двигается и не выходит за границы экрана.

## 📖 ЗАНЯТИЕ 2: ДВИЖЕНИЕ — ШТУРВАЛ

### 🎯 Цель

Освоить платформенное движение и управление.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Платформер! Гравитация, прыжки, платформы. Termix становится настоящим героем!»
- 5–15 мин: Platform Movement Object. Добавление движения.
- 15–30 мин: Настройка гравитации, скорости, прыжка.
- 30–45 мин: Создание платформ. Backdrop объекты.
- 45–55 мин: Termix бегает и прыгает по платформам.
- 55–60 мин: Итоги. Бадж «Платформер».

### 🛠️ Практика

Ребёнок добавляет Platform Movement Object к Termix'у. Настраивает: Gravity = 10, Jump Strength = 20, Speed = 30. Рисует платформы (Backdrop объекты). Termix автоматически стоит на платформах и прыгает при нажатии пробела.

### 🎮 Игровая механика

Соревнование «Кто выше прыгнет?». Дети меняют Jump Strength. Чей Termix допрыгнет до самой высокой платформы — жетон.

### 🏠 Миссия

- 🥉 Юнга: Добавить платформенное движение.
- 🥈 Боцман: Построить 3 платформы разной высоты.
- 🥇 Штурман: Сделать движущуюся платформу.

## 📖 ЗАНЯТИЕ 3: СТОЛКНОВЕНИЯ — РИФЫ

### 🎯 Цель

Освоить коллизии, сбор предметов и счёт.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Монетки! Золото! Termix собирает сокровища. Каждое касание — плюс очко!»
- 5–15 мин: Создание объекта-монетки.
- 15–30 мин: Событие: When Termix collides with Coin → Destroy Coin, Add 1 to Score.
- 30–45 мин: Создание счётчика. Объект Score или Counter.
- 45–55 мин: Условие победы: Score = 10 → Переход на следующий фрейм.
- 55–60 мин: Итоги. Бадж «Золотоискатель».

### 🛠️ Практика

Ребёнок создаёт объект Coin (золотой круг). Расставляет 10 монет на уровне. В Event Editor добавляет событие: Collision between Termix and Coin → Destroy Coin, Add 1 to Score. Создаёт счётчик. Условие: Score >= 10 → Next Frame.

### 🎮 Игровая механика

Соревнование «Кто быстрее соберёт 10?». Одинаковый уровень. Кто первый — жетон.

### 🏠 Миссия

- 🥉 Юнга: Собрать монетку (исчезает при касании).
- 🥈 Боцман: Добавить счётчик.
- 🥇 Штурман: Сделать монетки разной ценности (золото +2, серебро +1).

## 📖 ЗАНЯТИЕ 4: ВРАГИ — АБОРДАЖ

### 🎯 Цель

Освоить создание врагов, здоровье и условия поражения.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Враги! Крабы, акулы, пираты-конкуренты. Они двигаются и атакуют. Защищаемся!»
- 5–15 мин: Создание врага. Добавление движения (Path Movement или Bouncing Ball).
- 15–30 мин: Событие: When Termix collides with Enemy → Subtract 1 from Health.
- 30–45 мин: Здоровье. Счётчик здоровья. Условие: Health = 0 → Game Over.
- 45–55 мин: Неуязвимость после удара (таймер 2 секунды).
- 55–60 мин: Итоги. Бадж «Защитник».

### 🛠️ Практика

Ребёнок создаёт объект Enemy (красный краб). Добавляет Bouncing Ball Movement (двигается туда-сюда). Событие: Collision between Termix and Enemy → Subtract 1 from Health. Если Health = 0 → Jump to Game Over Frame.

### 🎮 Игровая механика

Игра «Выживи». 3 врага, 10 монет, 3 жизни. Кто продержится дольше всех — жетон.

### 🏠 Миссия

- 🥉 Юнга: Создать одного врага.
- 🥈 Боцман: Добавить здоровье и Game Over.
- 🥇 Штурман: Добавить неуязвимость после удара.

## 📖 ЗАНЯТИЕ 5: АНИМАЦИЯ — ОЖИВЛЕНИЕ

### 🎯 Цель

Освоить анимацию спрайтов.

### ⏱️ Тайминг (60–75 минут)

- 0–5 мин: Termix: «Картинка — скучно. Анимация — живо! Termix должен махать хвостом и моргать!»
- 5–15 мин: Импорт спрайт-листа. Настройка кадров анимации.
- 15–30 мин: Анимация покоя (Idle), бега (Run), прыжка (Jump).
- 30–45 мин: События смены анимации. When Termix is moving → Change animation to Run.
- 45–55 мин: Эффекты. Вспышка при получении урона.
- 55–60 мин: Итоги. Бадж «Аниматор».

### 🛠️ Практика

Ребёнок импортирует спрайт-лист Termix'а (4 кадра: idle, run1, run2, jump). В редакторе анимаций настраивает последовательности. Событие: When Termix is moving → Change animation to Run. When Termix is not moving → Change animation to Idle. When Termix is jumping → Change animation to Jump.

### 🎮 Игровая механика

Конкурс «Самая смешная анимация». Дети анимируют Termix'а. Голосование стикерами.

### 🏠 Миссия

- 🥉 Юнга: Сделать анимацию Idle.
- 🥈 Боцман: Добавить Run и Jump.
- 🥇 Штурман: Добавить анимацию получения урона.

- When Termix collides with Enemy → Play Sample hurt.wav.
- Start of Frame → Play Music pirate-music.mp3 (loop).
