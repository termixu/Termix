## 🧠 ФИЛОСОФИЯ КУРСА

Обычный подход: PHP → синтаксис → «я не понимаю, зачем это нужно» → скучные примеры.

Подход Termix Academy: PHP → серверные заклинания → форма обратной связи → гостевая книга → «я сделал сайт, который запоминает пользователей!».

PHP — это язык, на котором работает больше 75% всех сайтов в интернете (включая WordPress, Facebook в начале своего пути, Wikipedia). Termix учит детей превращать статичные HTML-страницы в динамические веб-приложения: обрабатывать формы, хранить данные в базе, выводить контент из базы данных.

📖 ЗАНЯТИЕ 2: ФОРМЫ — ПОСЛАНИЯ

🎯 Цель

Освоить GET и POST запросы.

⏱️ Тайминг (75–90 минут)

· 0–5 мин: Termix: «Формы! Пираты пишут послания. GET — крикнуть всем. POST — передать тайно. Учимся!»
· 5–15 мин: GET. Передача данных в URL. $_GET.
· 15–30 мин: POST. Скрытая передача. $_POST.
· 30–45 мин: HTML-форма. action, method.
· 45–60 мин: Обработка формы: приветствие по имени.
· 60–75 мин: Валидация: проверка пустого поля.
· 75–90 мин: Итоги. Бадж «Посланник».

🛠️ Практика

```php
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = $_POST["name"];
    if (!empty($name)) {
        echo "Бип! Привет, $name! Добро пожаловать в экипаж!";
    } else {
        echo "Ты забыл представиться, юнга!";
    }
}
?>

<form method="POST">
    <input type="text" name="name" placeholder="Твоё пиратское имя">
    <button type="submit">Взойти на борт!</button>
</form>
```

📖 ЗАНЯТИЕ 4: ФАЙЛЫ — БОРТОВОЙ ЖУРНАЛ

🎯 Цель

⏱️ Тайминг (75–90 минут)

· 0–5 мин: Termix: «Бортовой журнал! Записываем приключения в файл. Вечная память!»
· 5–15 мин: file_get_contents, file_put_contents.
· 15–30 мин: fopen, fwrite, fclose, fread.
· 30–45 мин: Запись сообщения в файл.
· 45–60 мин: Чтение всех сообщений и вывод.
· 60–75 мин: Форма для добавления записи в журнал.
· 75–90 мин: Итоги. Бадж «Летописец».

🛠️ Практика

```php
<?php
$logFile = "logbook.txt";

if ($_SERVER["REQUEST_METHOD"] == "POST" && !empty($_POST["entry"])) {
    $entry = date("Y-m-d H:i:s") . " — " . $_POST["entry"] . "\n";
    file_put_contents($logFile, $entry, FILE_APPEND);
}

echo "<h2>Бортовой журнал</h2>";

if (file_exists($logFile)) {
    $entries = file($logFile);
    foreach (array_reverse($entries) as $line) {
        echo "<p>$line</p>";
    }
}
?>

<form method="POST">
    <textarea name="entry" placeholder="Новая запись..."></textarea>
    <button type="submit">Записать</button>
</form>
```

📖 ЗАНЯТИЕ 5: MYSQL — БАЗА СОКРОВИЩ

🎯 Цель

Подключиться к MySQL и выполнить SELECT.

⏱️ Тайминг (75–90 минут)

· 0–5 мин: Termix: «База данных! Сокровища хранятся не в файлах, а в MySQL. Подключаемся!»
· 5–15 мин: phpMyAdmin. Создание базы данных и таблицы.
· 15–30 мин: Подключение: mysqli_connect.
· 30–45 мин: SELECT. Вывод данных на страницу.
· 45–60 мин: WHERE. Фильтрация.
· 60–75 мин: Практика. Таблица экипажа из базы данных.
· 75–90 мин: Итоги. Бадж «Хранитель базы».

🛠️ Практика

```php
<?php
$conn = mysqli_connect("localhost", "root", "", "pirate_db");

if (!$conn) {
    die("Ошибка подключения: " . mysqli_connect_error());
}

$result = mysqli_query($conn, "SELECT * FROM crew");

echo "<h2>Экипаж из базы данных</h2>";
echo "<table border='1'><tr><th>Имя</th><th>Звание</th><th>Золото</th></tr>";

while ($row = mysqli_fetch_assoc($result)) {
    echo "<tr>";
    echo "<td>{$row['name']}</td>";
    echo "<td>{$row['rank']}</td>";
    echo "<td>{$row['gold']}</td>";
    echo "</tr>";
}

echo "</table>";

mysqli_close($conn);
?>
```

Соревнование «Кто быстрее выведет?». Наставник: «Выведи всех капитанов из базы». Дети добавляют WHERE.

📖 ЗАНЯТИЕ 6: CRUD — УПРАВЛЕНИЕ

🎯 Цель

Освоить INSERT, UPDATE, DELETE.

⏱️ Тайминг (75–90 минут)

· 0–5 мин: Termix: «CRUD! Create, Read, Update, Delete. Полный контроль над данными!»
· 5–15 мин: INSERT. Добавление записи через форму.
· 15–30 мин: UPDATE. Изменение записи.
· 30–45 мин: DELETE. Удаление записи.
· 45–60 мин: Практика. Форма добавления пирата.
· 60–75 мин: Ссылки «Редактировать» и «Удалить».
· 75–90 мин: Итоги. Бадж «Капитан данных».

🛠️ Практика

```php
<?php
$conn = mysqli_connect("localhost", "root", "", "pirate_db");

// Добавление
if (isset($_POST['add'])) {
    $name = $_POST['name'];
    $rank = $_POST['rank'];
    $gold = $_POST['gold'];
    mysqli_query($conn, "INSERT INTO crew (name, rank, gold) VALUES ('$name', '$rank', $gold)");
}

// Удаление
if (isset($_GET['delete'])) {
    $id = $_GET['delete'];
    mysqli_query($conn, "DELETE FROM crew WHERE id = $id");
}
?>

<form method="POST">
    <input name="name" placeholder="Имя">
    <input name="rank" placeholder="Звание">
    <input name="gold" type="number" placeholder="Золото">
    <button name="add">Добавить</button>
</form>
```

📖 ЗАНЯТИЕ 8: АВТОРИЗАЦИЯ — ПАРОЛЬ

🎯 Цель

Освоить сессии и хеширование паролей.

⏱️ Тайминг (75–90 минут)

· 0–5 мин: Termix: «Авторизация! Только капитан может удалять сообщения. Пароль — твой ключ!»
· 5–15 мин: session_start(). $_SESSION.
· 15–30 мин: Форма логина. Проверка пароля.
· 30–45 мин: password_hash и password_verify.
· 45–60 мин: Защита страницы удаления (только для авторизованных).
· 60–75 мин: Выход (session_destroy).
· 75–90 мин: Итоги. Бадж «Ключник».

🛠️ Практика

```php
<?php
session_start();

$correctPassword = password_hash("рыба", PASSWORD_DEFAULT);

if (isset($_POST['login'])) {
    if (password_verify($_POST['password'], $correctPassword)) {
        $_SESSION['loggedin'] = true;
    }
}

if (isset($_GET['logout'])) {
    session_destroy();
    header("Location: index.php");
}
?>
```

🛠️ Практика

// Защита от SQL-инъекций
$stmt = $conn->prepare("INSERT INTO crew (name, rank, gold) VALUES (?, ?, ?)");

// Защита от XSS
echo htmlspecialchars($message, ENT_QUOTES, 'UTF-8');
