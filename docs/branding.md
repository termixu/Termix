## 🐱 ЛОГОТИП

### Основной логотип

**Termix с треуголкой, кибер-глазом и крюком.** Смотрит вправо (в будущее). Подпись: «TERMIX ACADEMY».

- **Формат:** Горизонтальный (кот слева, текст справа).
- **Цвета:** Оранжевый (кот) + зелёный (#00ff41, акценты) + чёрный (фон).
- **Стиль:** High-end 8bit pixel art.

### Варианты логотипа

- **Полный:** кот + текст «TERMIX ACADEMY».
- **Краткий:** только кот (для стикеров, аватарок).
- **Текстовый:** только «TERMIX ACADEMY» пиксельным шрифтом (для документов).
- **Монохромный:** белый кот на чёрном фоне, чёрный кот на белом фоне.

### Охранное поле

Вокруг логотипа должно быть свободное пространство, равное высоте буквы «T» в слове «TERMIX». Никакие элементы не должны заходить в это поле.

### Запрещено

- Растягивать или сжимать логотип.
- Менять цвета кота (оранжевый — священен).
- Ставить логотип на пёстрый фон (только тёмный или светлый).
- Использовать текстовый логотип без кота на продуктах для детей.

## 🎨 ЦВЕТА

### Основная палитра

- **Фирменный зелёный (Neon Green):** `#00ff41`. Используется для акцентов, кнопок, хвоста Termix'а, заголовков.
- **Глубокий чёрный (Dark Void):** `#0a0a0a`. Фон сайта, презентаций, терминала.
- **Тёплый оранжевый (Termix Orange):** `#ff6b35`. Цвет кота, основной тепло-акцент.
- **Золотой (Pirate Gold):** `#ffd700`. Баджи, жетоны, важные выделения.
- **Белый (Cloud):** `#ffffff`. Основной текст на тёмном фоне.

### Дополнительная палитра

- **Тёмно-зелёный (Deep Green):** `#0d1f0d`. Второстепенный фон, карточки.
- **Серый (Fog):** `#555555`. Хвост Termix'а в спящем режиме, неактивные элементы.
- **Красный (Alert Red):** `#ff0040`. Только для тревог и ошибок (Авария).
- **Синий (Processing Blue):** `#0099ff`. Только для состояния загрузки (Синий хвост).
- **Фиолетовый (Magic Purple):** `#bf00ff`. Только для особых моментов (праздник, секрет).

### Где что использовать

- **Сайт / Платформа:** фон `#0a0a0a`, текст `#ffffff`, акценты `#00ff41`, кнопки `#ff6b35`.
- **Презентации:** фон тёмный, заголовки `#00ff41`, текст `#ffffff`.
- **Документы:** фон белый, заголовки `#0a0a0a`, акценты `#00ff41`.
- **Стикеры:** оранжевый кот, зелёные полоски, чёрная шляпа.

## 🔤 ШРИФТЫ

### Основной шрифт (для документов, сайта, презентаций)

- **Press Start 2P** — пиксельный шрифт. Идеально для заголовков и логотипа.
- **Inter** — современный sans-serif. Для основного текста, интерфейса.

### Правила использования

- **Заголовки:** Press Start 2P, цвет `#00ff41` или `#ffffff`.
- **Подзаголовки:** Press Start 2P или Inter Bold.
- **Основной текст:** Inter Regular, 16–18px, цвет `#ffffff` на тёмном фоне.
- **Код:** Fira Code или JetBrains Mono, цвет `#00ff41` на `#0a0a0a`.

### Запрещено

- Использовать Comic Sans (никогда).
- Использовать более 2 шрифтов на одной странице.
- Растягивать Press Start 2P (он пиксельный, должен оставаться чётким).

## 🖼️ ВИЗУАЛЬНЫЙ СТИЛЬ

### High-end 8bit Pixel Art

Все изображения Termix Academy создаются в этом стиле:

- Чёткие пиксели без сглаживания.
- Объёмное неоновое освещение.
- Тёмные фоны с яркими акцентами.
- 16-битная глубина цвета.

### Принципы композиции

- Главный герой (Termix) всегда в центре внимания.
- Эмоции — преувеличенные, мультяшные.
- Взаимодействие с технологиями — игровое, весёлое.
- Никакой мрачности. Даже в тёмной серверной Termix улыбается.

### Где брать изображения

- Все изображения генерируются по утверждённым промптам (см. `termix-visual-style.md`).
- Никаких стоковых фото.
- Никаких чужих иллюстраций без лицензии.

## 🎖️ БАДЖИ И ЖЕТОНЫ

### Система баджей

Каждый бадж — это пиксельная иконка 256x256px в стиле Termix Academy:
- Тёмный фон с золотой рамкой.
- Центральный символ (связан с достижением).

- 🥇 **Штурман:** золотая рамка.
