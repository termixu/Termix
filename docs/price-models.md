## 🧠 ФИЛОСОФИЯ ЦЕНООБРАЗОВАНИЯ

Обычный подход: «Курс по Python — 500 рублей. Лишь бы записались». → Демпинг, выгорание, «дешёвый значит некачественный».

Подход Termix Academy: «Мы не продаём курсы. Мы продаём будущее ребёнка в IT. Кот-пират, экосистема, сообщество, результат».

Цена строится не из себестоимости (аренда, время), а из ценности для родителя и ребёнка. Родитель платит не за «10 занятий по Scratch», а за «ребёнок создал свою игру, поверил в себя и хочет стать программистом».

## 📊 ОБЗОР МОДЕЛЕЙ

### 1. Разовая оплата за курс

**Как работает:** Родитель платит один раз за полный курс (8–12 занятий). Доступ к материалам навсегда.

**Плюсы:**
- Простота: заплатил — получил.
- Предсказуемый доход.
- Легко считать юнит-экономику.

**Минусы:**
- Нет регулярного дохода.
- Каждый месяц нужно искать новых учеников.

**Пример цен:**
- Scratch (10 занятий): 4 900 руб.
- Python (12 занятий): 6 900 руб.
- Godot (12 занятий): 7 900 руб.
- Blender (12 занятий): 7 900 руб.

**Для кого:** Для родителей, которые хотят «попробовать» или «занять ребёнка на каникулах».

---

### 2. Подписка (Monthly / Annual)

**Как работает:** Родитель платит фиксированную сумму каждый месяц. Ребёнок получает доступ ко всем курсам, новым урокам, сообществу.

**Плюсы:**
- Регулярный предсказуемый доход (MRR).
- Удерживает учеников надолго.
- Стимулирует создавать новый контент.

**Минусы:**
- Сложнее продать (страх «вечной подписки»).
- Нужно постоянно давать ценность, чтобы не отписались.

**Пример цен:**
- Базовый тариф (все курсы, сообщество): 1 990 руб/мес.
- Премиум (всё + проверка домашек + личные консультации): 3 990 руб/мес.
- Годовой (скидка 20%): 19 000 руб/год (1 583 руб/мес).

**Для кого:** Для родителей, которые хотят, чтобы ребёнок рос в IT постоянно, а не «один курс и забыл».

---

### 3. Freemium + Платный контент

**Как работает:** Часть курсов — бесплатно. Продвинутые модули, проверка заданий, сертификаты — за деньги.

**Плюсы:**
- Огромный охват (бесплатное привлекает).
- Вирусный рост.
- Лёгкий вход для сомневающихся.

**Минусы:**
- Бесплатные пользователи не платят.
- Низкая конверсия в платное (обычно 2–5%).

**Пример:**
- Бесплатно: Первые 3 занятия любого курса.
- Платно: Полный курс (4 900–7 900 руб).
- Бесплатно: Терминал Termius (базовые команды).
- Платно: Полный сериал + квесты (299 руб/мес).

**Для кого:** Для привлечения новой аудитории через соцсети, YouTube, рилсы.

---

### 4. Пакеты и Специализации

**Как работает:** Объединение нескольких курсов в пакет со скидкой. Например, «Python-разработчик» = Python + Git + VS Code + SQL + Flask.

**Плюсы:**
- Выше средний чек.
- Удерживает ученика на 6–12 месяцев.
- Формирует профессиональные треки.

**Минусы:**
- Сложнее продать (дороже).
- Нужно, чтобы все курсы в пакете были готовы.

**Примеры пакетов:**
- 🎮 **Гейм-дев:** Scratch → Kodu → Roblox → Godot → Unreal Engine 4. Цена: 24 900 руб (экономия 30%).
- 🐍 **Python-разработчик:** Python → Git → VS Code → SQL → Flask → Docker. Цена: 29 900 руб (экономия 30%).
- 🎨 **3D-Художник:** Tinkercad → Piskel → Spline → Blender → Figma → FAB. Цена: 29 900 руб (экономия 30%).
- 🧠 **Fullstack-пират:** HTML → CSS → JavaScript → React → Node.js → Docker → GitHub Actions. Цена: 34 900 руб (экономия 30%).

**Для кого:** Для мотивированных родителей, которые хотят дать ребёнку полноценную профессию.

### 5. Клубная система (Membership)

**Как работает:** Закрытое сообщество с ежемесячной подпиской. Включает: доступ к Discord-серверу Termix Academy, еженедельные стримы с Termix'ом, разбор домашек, Code Review, хакатоны, общение с другими юнгами.

**Плюсы:**
- Высокая лояльность.
- Дети дружат и растут вместе.
- Родители видят сообщество и не уходят.

**Минусы:**
- Требует времени на модерацию и проведение стримов.
- Нужна критическая масса участников для активности.

**Пример цен:**
- Юнга (доступ к Discord, чатам): 490 руб/мес.
- Боцман (+ еженедельные стримы и разбор домашек): 990 руб/мес.
- Капитан (+ личная консультация Termix'а раз в месяц): 2 490 руб/мес.

**Для кого:** Для детей, которые уже прошли курсы и хотят продолжать расти в окружении единомышленников.

### 6. B2B: Продажа методик школам и лагерям

**Как работает:** Termix Academy продаёт готовые программы, методички, обучение преподавателей для школ, кружков и IT-лагерей.

**Плюсы:**
- Крупные чеки.
- Масштабирование без увеличения нагрузки на Артёма.
- Партнёрства с образовательными учреждениями.

**Минусы:**
- Сложный цикл продаж (тендеры, согласования).
- Нужна юридическая упаковка.

**Пример цен:**
- Коробка «Scratch + Termix» (программа, методичка, слайды, баджи): 29 000 руб (единоразово).
- Обучение преподавателя (2 дня, онлайн): 19 000 руб.
- Франшиза Termix Academy (бренд, методики, поддержка): от 50 000 руб/мес.

**Для кого:** Для образовательных учреждений, которые хотят запустить IT-направление.

### 7. Маркетплейс (Пассивный доход)

**Как работает:** Ученики Termix Academy создают цифровые продукты и продают их на платформах. Termix Academy получает комиссию или продвигает своих учеников.

**Плюсы:**
- Пассивный доход.
- Ученики зарабатывают и становятся адвокатами бренда.
- Реальные кейсы для маркетинга.

**Минусы:**
- Доход зависит от успеха учеников.
- Может отвлекать от основного обучения.

**Пример:**

- Ученик создал 3D-модель → продал на FAB за $10.
- Termix Academy продвигает модели учеников в своих соцсетях.
- Партнёрская программа: ученик получает заказ через Академию → 10% комиссия платформе.

**Для кого:** Для продвинутых учеников 14+ лет, которые хотят монетизировать навыки.

---

### 8. Стипендии и Гранты (Социальная миссия)

**Как работает:** Termix Academy выделяет 1–2 бесплатных места в каждом потоке для талантливых детей из малообеспеченных семей. Финансируется за счёт платных учеников или спонсоров.

**Плюсы:**
- Социальная миссия.
- PR и репутация.
- Мотивирует платных учеников («я помогаю другим учиться»).

**Минусы:**
- Уменьшает прибыль.
- Нужен отбор (критерии).

**Пример:**

- 1 бесплатное место на каждые 10 платных.
- Заявки через форму: рассказать, почему хочешь учиться.
- Спонсоры (IT-компании) оплачивают стипендии в обмен на упоминание.

**Для кого:** Для создания позитивного имиджа и привлечения талантов.
