## 🎯 ЧТО ДЕЛАЕТ VIBE CODER

Vibe Coder — это программист новой эры. Он не сидит часами над синтаксисом, не зубрит методы и не боится пустого файла. Он объясняет задачу ИИ, проверяет результат, собирает из кусков работающее приложение и кайфует от процесса.

Это профессия, которая появилась в 2024–2025 годах вместе с мощными AI-инструментами для кода. И она набирает обороты быстрее любой другой IT-специальности.

**Типичные задачи:**
- Описать приложение словами и получить работающий код.
- Собрать сайт, игру или бота через промпты.
- Проверить сгенерированный код и поправить ошибки.
- Связать фронтенд и бэкенд, даже если не знаешь всё досконально.
- Задеплоить готовый продукт.
- Продолжать учиться, потому что ИИ помогает, но не заменяет понимание.

**Где работает:**
- В стартапах (быстрое прототипирование).
- На фрилансе (сборка проектов под ключ).
- Как инди-разработчик (свои продукты).
- В любой компании, где ценят скорость.

Vibe Coder — это не замена классическому программисту. Это ускоритель. Сначала нужно понять базу. Потом — научиться управлять ИИ. Потом — летать.

### Этап 1: Фундамент (12–13 лет)

Курсы Termix Academy, которые дают понимание, как работает код:

- **AI-промптинг** — основной курс. Как общаться с ИИ, давать роли, писать спецификации, получать код, отлаживать его.

- **HTML + CSS** — понимание структуры веб-страниц.

- **Python** — основы программирования. Переменные, функции, библиотеки.

- **VS Code** — редактор, в котором живёт Copilot.

- **Git и GitHub** — сохранение версий своего кода.

**Что умеет после этапа:**
- Написать спецификацию для генерации кода.
- Получить от ИИ рабочий скрипт или страницу.
- Прочитать код и понять, что он делает.

**Первые проекты для портфолио:**
- Сайт-визитка, собранный через промпты.
- Python-скрипт, сгенерированный ИИ и проверенный тобой.
- Коллекция удачных промптов для кода.

### Этап 2: Ускорение (13–15 лет)

- **Copilot** — ИИ-ассистент в редакторе. Спецификации, автодополнение, чат, отладка.

- **JavaScript** — язык браузера. Оживление страниц.

- **React** — компонентный подход. Сборка интерфейсов.

- **Flask (Python)** — бэкенд для веб-приложений.

- **SQL и PostgreSQL** — базы данных.

- **Docker** — упаковка приложений.

- **GitHub Actions** — автоматический деплой.

**Что умеет после этапа:**
- Собрать SPA на React через промпты.
- Написать API на Flask через спецификации.
- Связать фронтенд и бэкенд.
- Задеплоить приложение.

**Первые проекты для портфолио:**
- Fullstack-приложение (React + Flask + PostgreSQL), собранное с Copilot.
- Telegram-бот с GenAI.
- Мини-игра на JavaScript, сгенерированная по спецификации.
