## 🎯 ЧТО ДЕЛАЕТ FRONTEND DEVELOPER

Frontend Developer — это мост между дизайном и кодом. Он берёт макет из Figma и превращает его в живую страницу. Кнопки нажимаются, формы отправляются, анимации проигрываются, данные подгружаются с сервера.

**Типичные задачи:**
- Сверстать страницу по макету (HTML + CSS).
- Добавить интерактивность (JavaScript).
- Связать интерфейс с сервером (API).
- Сделать адаптив: чтобы сайт работал на телефоне, планшете, ноутбуке.
- Оптимизировать скорость загрузки.
- Писать тесты на интерфейс.

**Где работает:**
- В IT-компаниях (от стартапов до корпораций).
- На фрилансе (заказы на сайты и приложения).
- В игровых студиях (интерфейсы игр).
- Удалённо из любой точки мира.

## 🗺️ ПУТЬ В TERMIX ACADEMY

### Этап 1: Фундамент (11–13 лет)

Курсы Termix Academy, с которых начинается путь:

- **HTML** — каркас сайта. Заголовки, параграфы, списки, таблицы, формы, семантические теги.

- **CSS** — стилизация. Цвета, шрифты, отступы, Flexbox, Grid, анимации, адаптив.

- **VS Code** — редактор кода. Расширения, Emmet, хоткеи, Prettier, Live Server.

- **Git и GitHub** — контроль версий. Сохранять историю изменений, работать в команде, выкладывать проекты.

**Что умеет после этапа:**
- Сверстать страницу по макету.
- Сделать адаптив (мобильная версия).
- Выложить сайт на GitHub Pages.

**Первые проекты для портфолио:**
- Одностраничный сайт «Пиратский штаб».
- Галерея карточек с Flexbox.
- Адаптивная страница с формой.

---

### Этап 2: Интерактивность (13–15 лет)

- **JavaScript** — язык браузера. События, DOM, массивы, объекты, запросы к API, Canvas.

- **Terminal (Bash)** — командная строка для сборки проектов, запуска скриптов, работы с npm.

- **Figma** — чтение макетов. Фронтендер не рисует, но должен понимать дизайн.

**Что умеет после этапа:**
- Добавить интерактивность на сайт (клики, формы, анимации).
- Получить данные с сервера и отобразить на странице.
- Собрать простую игру на Canvas.

**Первые проекты для портфолио:**
- Браузерная игра «Змейка-Термикс».
- Сайт-портфолио с контактной формой.
- Приложение «Список пиратов» (добавление, удаление, фильтрация).

---

### Этап 3: Профессиональный инструментарий (15–16 лет)

- **React** — главная библиотека фронтенда. Компоненты, хуки, состояние, роутинг, контекст.

- **GitHub Actions** — автоматический деплой сайта при пуше.

- **Docker** — контейнеризация. Запуск фронтенда в изолированной среде.

- **Фриланс** — первые заказы на вёрстку и React-компоненты.

**Что умеет после этапа:**
- Собрать SPA (Single Page Application) на React.
- Настроить автодеплой.
- Взять заказ и довести до сдачи.

**Проекты для портфолио:**
- Приложение Termix Academy на React (многостраничное).
- Дашборд с графиками (данные из API).
- Коммерческий проект (реальный заказ).

## 🛠️ ИНСТРУМЕНТЫ FRONTEND DEVELOPER

### Основной стек

- **HTML + CSS** — база. Фронтендер должен знать их идеально.
- **JavaScript** — язык интерактивности.
- **React** — главный фреймворк (или Vue, Angular).
- **Git и GitHub** — система контроля версий.
- **VS Code** — редактор кода.

### Инструменты сборки

- **Vite** — сборщик проектов (быстрый, современный).
- **npm** — менеджер пакетов.
- **ESLint + Prettier** — автоформатирование и проверка кода.

### Для работы с дизайном

- **Figma** — просмотр макетов.
- **DevTools (F12)** — отладка прямо в браузере.

### Где брать заказы

- **Fiverr** — международная биржа. Заказы от $50.
- **Kwork** — российская биржа. Кворки от 1,500 руб.
- **Upwork** — крупные контракты ($500+).
- **LinkedIn** — профессиональная сеть, прямые работодатели.

### Где показывать работы

- **GitHub** — главное портфолио фронтендера.
- **LinkedIn** — профессиональный профиль.
- **Личный сайт** — витрина своих проектов.

## 🏆 ПРИМЕРЫ ПРОЕКТОВ ДЛЯ ПОРТФОЛИО

### Уровень Senior (после полного трека)

- Коммерческий проект, работающий в продакшене.
- Свой npm-пакет (опубликованная библиотека).
- Профиль на GitHub с 100+ звёзд.
