## 🎯 ЧТО ДЕЛАЕТ BACKEND DEVELOPER

Backend Developer — это инженер невидимого фронта. Когда пользователь нажимает кнопку «Купить», фронтенд отправляет запрос, а бэкенд проверяет баланс, списывает деньги, обновляет базу и отправляет ответ. Всё это за доли секунды.

**Типичные задачи:**
- Спроектировать базу данных (таблицы, связи).
- Написать API (интерфейс для общения фронтенда с сервером).
- Обработать данные от пользователя и проверить их.
- Настроить авторизацию и безопасность.
- Оптимизировать запросы, чтобы сервер не падал под нагрузкой.
- Писать тесты на логику.

**Где работает:**
- В IT-компаниях (от стартапов до FAANG).
- На фрилансе (заказы на серверную часть).
- В банках, телекоме, ритейле — везде, где есть данные.
- Удалённо из любой точки мира.

## 🗺️ ПУТЬ В TERMIX ACADEMY

### Этап 1: Фундамент (12–14 лет)

Курсы Termix Academy, с которых начинается путь:

- **Python** — главный язык для входа в бэкенд. Переменные, функции, циклы, работа с файлами, библиотеки.

- **Terminal (Bash)** — командная строка. Бэкендер живёт в терминале.

- **VS Code** — редактор кода. Расширения для Python, отладка.

- **Git и GitHub** — контроль версий. Без этого не берут ни в одну компанию.

**Что умеет после этапа:**
- Писать скрипты на Python.
- Работать с файлами и библиотеками.
- Понимать, что такое сервер и клиент.

**Первые проекты для портфолио:**
- Консольная игра «Текстовый квест».
- Скрипт для анализа логов.
- Генератор пиратских имён.

---

### Этап 2: Базы данных и API (13–15 лет)

- **SQL и PostgreSQL** — язык запросов к базам данных. SELECT, JOIN, GROUP BY, индексы.

- **Flask (Python)** — веб-фреймворк. Создание API, обработка запросов, маршруты.

- **Linux** — операционная система для серверов. Большинство серверов в мире работают на Linux.

- **Jupyter** — среда для экспериментов с данными и Python.

**Что умеет после этапа:**
- Спроектировать базу данных из 3–5 таблиц.
- Написать API с эндпоинтами (GET, POST, PUT, DELETE).
- Поднять сервер на Flask.

**Первые проекты для портфолио:**
- API для пиратского приложения (пираты, корабли, сокровища).
- Бот для Telegram на Python + SQLite.
- Дашборд с аналитикой (данные из базы).

---

### Этап 3: Профессиональный инструментарий (15–16 лет)

- **Docker** — контейнеризация. Упаковка приложения со всеми зависимостями.

- **GitHub Actions** — автоматические тесты и деплой.

- **Node.js** — второй язык для бэкенда (JavaScript на сервере).

- **Фриланс** — первые заказы на API и базы данных.

**Что умеет после этапа:**
- Упаковать приложение в Docker.
- Настроить CI/CD (автотесты + автодеплой).
- Работать с двумя языками (Python + Node.js).

**Проекты для портфолио:**
- API с авторизацией (JWT), упакованный в Docker.
- Микросервис для Termix Academy.
- Коммерческий проект (реальный заказ).
