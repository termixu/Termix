## 🎯 ЧТО ДЕЛАЕТ WEB DESIGNER

Web Designer — это архитектор визуального интернета. Каждый сайт, который ты открываешь, кто-то спроектировал: выбрал цвета, расставил кнопки, подобрал шрифты, продумал, куда пойдёт взгляд пользователя.

**Типичные задачи:**
- Нарисовать макет сайта в Figma.
- Подобрать цветовую схему и шрифты.
- Спроектировать удобную навигацию.
- Сделать адаптив: чтобы сайт хорошо выглядел и на компьютере, и на телефоне.
- Подготовить иконки и изображения.
- Передать макет разработчику.

**Где работает:**
- В веб-студиях и digital-агентствах.
- На фрилансе (заказы от бизнеса).
- В IT-компаниях (внутренние продукты).
- Как независимый специалист (свои клиенты).

## 🗺️ ПУТЬ В TERMIX ACADEMY

### Этап 1: Основы визуала (10–12 лет)

Курсы Termix Academy, с которых начинается путь:

- **HTML** — каркас сайта. Дизайнер должен понимать, из чего состоит страница: заголовки, параграфы, секции, кнопки. Не обязательно верстать идеально, но нужно знать возможности HTML.

- **CSS** — язык стилей. Цвета, шрифты, отступы, тени, анимации. Дизайнер, который знает CSS, говорит с разработчиком на одном языке.

- **Piskel** — пиксель-арт редактор. Создание простых иконок и графических элементов для сайта.

**Что умеет после этапа:**
- Сверстать простую страницу.
- Подобрать цвета и шрифты.
- Нарисовать иконку.
- Понимать, как HTML и CSS работают вместе.

**Первые проекты для портфолио:**
- Одностраничный сайт «Пиратский штаб» (свёрстан и оформлен).
- Набор иконок для сайта (меню, поиск, контакты).
- Палитра цветов Termix Academy (документ с цветами и шрифтами).

---

### Этап 2: Дизайн и прототипы (12–14 лет)

- **Figma** — главный инструмент веб-дизайнера. Макеты, компоненты, авто-лейауты, прототипы.

- **Spline** — 3D в браузере. Можно добавить на сайт интерактивную 3D-графику.

- **JavaScript (база)** — понимание того, что можно сделать на сайте. Анимации, всплывающие окна, переключение тем.

**Что умеет после этапа:**
- Нарисовать макет сайта в Figma.
- Создать дизайн-систему (компоненты, стили, сетка).
- Сделать кликабельный прототип.
- Понимать адаптив (мобильная версия, планшет, десктоп).

**Первые проекты для портфолио:**
- Макет сайта Termix Academy в Figma (3 страницы).
- Кликабельный прототип приложения.
- Ред дизайн существующего сайта (взять неудачный сайт и переделать).

---

### Этап 3: Профессиональный уровень (14–16 лет)

- **JavaScript (углублённо)** — анимации, параллакс, интерактивные элементы.

- **React** — понимание компонентного подхода. Дизайнер, знающий React, проектирует интерфейсы, которые легко сверстать.

- **Фриланс** — первые заказы. Лендинги, визитки, редизайн.

- **AI-промптинг** — генерация изображений, иконок и идей через нейросети.

**Что умеет после этапа:**
- Взять проект от брифинга до сдачи.
- Общаться с заказчиком и разработчиком.
- Делать дизайн, который удобно верстать.
- Оценивать свою работу в деньгах.

**Проекты для портфолио:**
- Многостраничный сайт (лендинг + блог + контакты).
- Коммерческий проект (реальный заказ).
- Дизайн-система в Figma (компоненты, сетки, стили).

## 🛠️ ИНСТРУМЕНТЫ WEB DESIGNER

### Для дизайна

- **Figma** — основной инструмент. Бесплатный для одного пользователя, работает в браузере.
- **Adobe XD** — альтернатива Figma (реже используется).
- **Spline** — 3D-графика для сайтов.

### Для графики

- **Piskel** — пиксельные иконки.
- **Canva** — быстрые баннеры и изображения.
- **Photoshop / GIMP** — редактирование фото.
- **DALL-E / Midjourney** — генерация изображений через промпты.

### Для прототипирования

- **Figma Prototype** — встроенный инструмент.
- **HTML + CSS** — живой прототип в браузере.

### Где брать заказы

- **Fiverr** — международная биржа. Заказы от $25.
- **Kwork** — российская биржа. Кворки от 500 руб.
- **Behance** — портфолио. Клиенты приходят сами.
- **Прямые клиенты** — местный бизнес, знакомые, соцсети.

### Где показывать работы

- **Behance** — главная платформа для дизайнеров.
- **Dribbble** — сообщество дизайнеров.
- **Instagram / Telegram** — личный бренд.
- **LinkedIn** — профессиональная сеть.

## 💰 СКОЛЬКО ЗАРАБАТЫВАЕТ WEB DESIGNER

### Работа в компании

- Junior Web Designer: $20,000–35,000 в год.
- Middle: $45,000–70,000 в год.
- Senior / Lead Designer: $80,000–110,000 в год.

### Фриланс

- Лендинг (одностраничный сайт): $200–800.
- Многостраничный сайт: $500–3,000.
- Ред дизайн: $150–500.
- Дизайн-система: $1,000–5,000.

### Реалистичный путь подростка

- 12–14 лет: первые макеты в Figma, помощь друзьям.
- 14–16 лет: первые заказы на Kwork (500–2,000 руб), портфолио на Behance.
- 16–18 лет: регулярный доход $300–1,000 в месяц, свои клиенты.

## 🏆 ПРИМЕРЫ ПРОЕКТОВ ДЛЯ ПОРТФОЛИО

- Свёрстанный и оформленный одностраничный сайт.

### Уровень Senior (после полного трека)

- Коммерческий проект с брифингом, макетом и сдачей.
- Дизайн-система в Figma.
- Профиль на Behance с 500+ просмотрами.
