## 🎯 ЧТО ДЕЛАЕТ DEVOPS ENGINEER

DevOps Engineer — это мост между разработкой и серверами. Он настраивает пайплайны: разработчик запушил код → GitHub Actions запустил тесты → Docker собрал образ → сервер обновился. Всё автоматически.

**Типичные задачи:**
- Настроить CI/CD (Continuous Integration / Continuous Deployment).
- Упаковать приложение в Docker-контейнер.
- Поднять сервер в облаке (AWS, Google Cloud).
- Настроить мониторинг (логи, алерты).
- Автоматизировать рутину bash-скриптами.
- Обеспечить безопасность инфраструктуры.

**Где работает:**
- В IT-компаниях любого размера.
- В стартапах (часто первый и единственный DevOps).
- На фрилансе (настройка серверов под ключ).
- Удалённо (серверы не требуют присутствия в офисе).
