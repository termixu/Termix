## 🎯 ЧТО ДЕЛАЕТ GAME DEVELOPER

Game Developer — это программист, который специализируется на играх. Он не просто пишет код, он создаёт миры, в которых живут игроки. От физики прыжка до поведения босса — всё это его работа.

**Типичные задачи:**
- Запрограммировать движение персонажа и физику.
- Написать поведение врагов (AI).
- Создать систему здоровья, очков и инвентаря.
- Собрать уровень из тайлов.
- Настроить анимации и эффекты.
- Оптимизировать игру, чтобы не тормозила.
- Подготовить сборку для публикации.

**Где работает:**
- В игровых студиях (от инди до AAA).
- Как инди-разработчик (свои игры).
- На фрилансе (движки, прототипы, ассеты).
- В смежных индустриях: VR, метавселенные, симуляторы.

## 🗺️ ПУТЬ В TERMIX ACADEMY

### Этап 1: Первые игры (10–12 лет)

Курсы Termix Academy, с которых начинается путь:

- **Scratch** — визуальное программирование игр. Переменные, условия, циклы, сообщения. Ребёнок создаёт кликер, лабиринт, догонялки.

- **Piskel** — рисование спрайтов. Разработчик должен понимать, как устроена графика.

- **HTML + CSS + JavaScript** — браузерные игры. Первый текстовый код. Canvas, анимации, обработка клавиш.

**Что умеет после этапа:**
- Создать простую игру в Scratch.
- Написать браузерную игру на JavaScript (змейка, кликер).
- Нарисовать спрайты для своего проекта.

**Первые проекты для портфолио:**
- Кликер «Корми Термикса» (Scratch).
- «Змейка-Термикс» (JavaScript + Canvas).
- Лабиринт с procedurally generated уровнями.

---

### Этап 2: Игровые движки (12–15 лет)

- **Godot** — бесплатный игровой движок. Сцены, ноды, GDScript, физика, анимации, UI. Основной инструмент курса.

- **Blender** — 3D-моделирование для игр. Персонажи, окружение, анимации.

- **VoxEdit** — воксельные модели. Идеально для игр в стиле Minecraft.

- **Python** — скрипты для инструментов, генерация данных, анализ игровой экономики.

**Что умеет после этапа:**
- Собрать 2D-платформер в Godot.
- Создать 3D-модели в Blender и импортировать в движок.
- Написать поведение врагов (AI).
- Настроить UI и звук.

**Первые проекты для портфолио:**
- Платформер «Termix: Pirate Adventure» в Godot.
- 3D-сцена «Остров сокровищ» (Blender + Godot).
- Игра с волнами врагов.

---

### Этап 3: Профессиональный уровень (15–16 лет)

- **Steam Workshop** — публикация контента для игр. Понимание, как устроен рынок.

- **FAB** — маркетплейс 3D-ассетов. Продажа моделей для чужих игр.

- **Docker + GitHub Actions** — автоматизация сборок.

- **Фриланс** — заказы на прототипы, движки, ассеты.

**Что умеет после этапа:**
- Собрать игру от идеи до публикации.
- Продавать ассеты на FAB.
- Работать с заказчиками.

**Проекты для портфолио:**
- Полноценная игра на itch.io.
- Набор ассетов на FAB.
- Коммерческий прототип для заказчика.

## 🛠️ ИНСТРУМЕНТЫ GAME DEVELOPER

- **Piskel** — пиксель-арт.
- **Blender** — 3D-моделирование.

### Платформы для публикации

- **itch.io** — инди-маркетплейс.
- **Steam Direct** — публикация в Steam ($100 за слот).
- **Steam Workshop** — контент для существующих игр.
- **FAB** — маркетплейс 3D-ассетов.

### Где брать заказы

- **Fiverr** — заказы на прототипы и ассеты от $50.
- **Unity Asset Store / FAB** — пассивный доход с продаж.
- **itch.io** — продажа своих игр.
