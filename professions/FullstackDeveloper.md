## 🎯 ЧТО ДЕЛАЕТ FULLSTACK DEVELOPER

Fullstack Developer — это универсальный солдат. Он может в одиночку собрать веб-приложение: сверстать интерфейс, написать серверную логику, настроить базу данных, задеплоить на сервер. Стартапы обожают фуллстеков: один человек вместо двух.

**Типичные задачи:**
- Сверстать страницу по макету (HTML, CSS, React).
- Написать API для передачи данных (Python/Flask или Node.js).
- Спроектировать базу данных (PostgreSQL).
- Связать фронтенд с бэкендом.
- Настроить авторизацию пользователей.
- Задеплоить приложение на сервер (Docker, облака).
- Писать тесты на всём стеке.

**Где работает:**
- В стартапах (часто первый и единственный разработчик).
- На фрилансе (берёт проект целиком).
- В IT-компаниях (работает над продуктом на всех уровнях).
- Как инди-разработчик (свои проекты).

## 🗺️ ПУТЬ В TERMIX ACADEMY

Фуллстек — это не отдельный курс. Это комбинация двух треков: фронтенд + бэкенд. Ребёнок сначала осваивает одну сторону, потом вторую, потом учится соединять их вместе.

### Этап 1: Фронтенд (13–14 лет)

Курсы Termix Academy для фронтенда:

- **HTML** — каркас страницы.
- **CSS** — стилизация, анимации, адаптив.
- **JavaScript** — интерактивность, события, Canvas.
- **Figma** — чтение макетов.
- **VS Code** — редактор кода.
- **Git и GitHub** — контроль версий.

**Что умеет после этапа:**
- Сверстать адаптивную страницу.
- Добавить интерактивность на JavaScript.
- Выложить сайт на GitHub Pages.

**Первые проекты для портфолио:**
- Игра «Змейка-Термикс» на Canvas.
- Сайт-портфолио с анимациями.

### Этап 2: Бэкенд (14–15 лет)

Курсы Termix Academy для бэкенда:

- **Python** — основной язык.
- **SQL и PostgreSQL** — базы данных.
- **Flask (Python)** — веб-фреймворк, API.
- **Terminal (Bash)** — командная строка.
- **Linux** — серверная ОС.

Курсы Termix Academy для соединения:
