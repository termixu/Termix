## 🧒 5–7 лет (Дошкольники)

- **ПиктоМир** — робот-пират, команды через пиктограммы, алгоритмы без текста.

---

## 👶 6–8 лет (Малыши)

- **Scratch Jr** — визуальное программирование, блоки как кубики, анимации и истории.
- **Kodu Game Lab** — 3D-миры без кода, правила WHEN-DO, острова и персонажи.
- **Delightex** — Лего-игры, сборка уровней из готовых блоков поведения.

---

## 👦 7–9 лет (Младшая группа — Начало)

- **Google Blockly** — головоломки, лабиринты, птицы, черепахи, визуальные алгоритмы.
- **LEGO Digital Designer** — цифровая верфь, бесконечные детали LEGO, 3D-модели.
- **Minecraft Education** — шахты знаний, Agent-помощник, Code Builder.

---

## 🧒 8–10 лет (Младшая группа — Основная)

- **Scratch** — визуальное программирование, игры, мультики, алгоритмы.
- **Tinkercad** — 3D-моделирование из блоков, подготовка к 3D-печати.
- **Paint.NET** — пиратская кисть, слои, эффекты, графика для проектов.
- **Clideo** — капитанский монтаж, онлайн-видеоредактор, обрезка и склейка.
- **Microsoft Office** — капитанская канцелярия, Word, Excel, PowerPoint.

---

## 👦 9–11 лет (Средняя группа — Начало)

- **PowerPoint** — флагманская презентация, слайды, анимация, выступление.
- **Animaker** — мульт-студия Termix'а, анимация без покадровой отрисовки.
- **Spline** — браузерный 3D-редактор, анимации, интерактивные сцены.
- **CapCut** — пиратская монтажная, рилсы, переходы, музыка, текст.
- **Piskel** — пиксель-арт редактор, создание спрайтов и анимаций.

---

## 🧑 10–12 лет (Средняя группа — Основная)

- **App Inventor** — карманный корабль, мобильные приложения без кода.
- **Thunkable** — карманный корабль 2.0, приложения для iOS и Android.
- **VoxEdit** — воксельное моделирование для Minecraft и игр.
- **Clickteam Fusion** — фабрика игр, игры без кода, событийная система.
- **Git и GitHub** — контроль версий, командная работа.
- **Visual Studio Code** — редактор кода, расширения, терминал.
- **Terminal (Bash)** — базовые команды, навигация по файловой системе.
- **HTML** — каркас пиратского корабля, вёрстка веб-страниц.
- **CSS** — окраска корабля, стили, Flexbox, анимации.

---

## 👨 11–13 лет (Старшая группа — Начало)

- **Python** — текстовое программирование, переменные, циклы, функции.
- **JavaScript** — программирование в браузере, DOM, события, Canvas.
- **Jupyter** — лаборатория заклинаний, интерактивные тетради, pandas, matplotlib.
- **Figma** — картографическая мастерская, дизайн интерфейсов, прототипирование.
- **SketchUp** — архитектурная бухта, 3D-здания, крепости, причалы.
- **Roblox Studio** — верфь миров, создание игр на Lua, публикация, Robux.
- **GIMP** — пиратская графика, слои, выделение, фильтры, логотипы.

---

## 👨 12–14 лет (Старшая группа — Основная)

- **Blender** — профессиональное 3D-моделирование, скульптинг, анимация, рендер.
- **Termius** — Ядро в кармане, мобильный терминал, сериал, квесты.
- **Adobe Photoshop** — мастерская цифровой магии, ретушь, маски, коллажи.
- **Godot** — игровой движок, создание 2D и 3D игр, GDScript.
- **React** — библиотека для создания интерфейсов, компоненты, хуки.
- **Flask (Python)** — веб-фреймворк, создание сайтов и API.
- **Node.js** — серверный JavaScript, создание API.
- **SQL и PostgreSQL** — базы данных, запросы, проектирование.

---

## 🎓 13–15 лет (Профи-группа — Начало)

- **Linux** — операционная система, управление сервером.
- **Docker** — контейнеризация, развёртывание приложений.
- **GitHub Actions** — автоматизация, CI/CD.
- **AI-промптинг** — промпт-инжиниринг, работа с LLM, генерация контента.
- **Copilot** — цифровой боцман, AI-ассистент, спецификации, отладка.
- **Telegram Bot API** — создание ботов на Python и JS.
- **Steam Workshop** — публикация игровых ассетов, скинов, стикеров.
- **FAB** — маркетплейс 3D-моделей, монетизация творчества.

---

## 🎓 14–16 лет (Профи-группа — Основная)

- **PHP** — серверные заклинания, формы, MySQL, CRUD, авторизация.
- **C#** — арсенал Unity, ООП, LINQ, коллекции, консольные RPG.
- **Java** — кодовая крепость, ООП, исключения, консольные приложения.
- **Unreal Engine 4** — кузница миров, Blueprints, ландшафт, AAA-графика.
- **Фриланс** — профиль, заказы, работа с клиентами.
- **C++** — машинное отделение, указатели, память, ООП.
