## 🎯 Миссия

**Termix Academy** — это экосистема обучения детей 10–14 лет современным IT-навыкам. Мы не читаем скучные лекции. Мы отправляем юнг в цифровое плавание, где каждый `git commit` — это запись в бортовом журнале, каждый баг — морское чудовище, а деплой — триумфальное возвращение в порт.

Наш проводник — **Termix**, кибер-пиратский кот с неоновым глазом и чёрной треуголкой. Он знает Python, JavaScript, Spline 3D и умеет объяснить `git rebase` через мем с попугаем.

## 🎮 Как устроено обучение

### Три возрастных экипажа

| Экипаж | Возраст | Что изучают | Главный проект |
|:---|:---|:---|:---|
| 🧒 **IT-Разведчики** | 10–11 лет | ПК-грамотность, Scratch, Spline 3D | Интерактивная 3D-сцена |
| 👦 **Кодовые Пираты** | 12–13 лет | HTML/CSS, Python Turtle, Git, JavaScript | Кликер «Корми Термикса» |
| 🧑 **IT-Архитекторы** | 14+ лет | Flask, React, SQL, Linux, AI | Свой AI-ассистент |

### Принципы обучения

| 🏴‍☠️ Принцип | 💡 Что это значит |
|:---|:---|
| **Геймификация** | Баджи, ранги, опыт, PvP-дуэли |
| **Мем-педагогика** | Сложное объясняется через мемы и шпаргалки |
| **Проектный подход** | Каждый модуль заканчивается проектом |
| **Termix AI** | Кот-наставник помогает и мотивирует |
| **Peer-to-peer** | Старшие ученики менторят младших |
