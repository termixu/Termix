## 🎯 Миссия

**Termix Academy** — это экосистема обучения детей 10–14 лет современным IT-навыкам. Мы не читаем скучные лекции. Мы отправляем юнг в цифровое плавание, где каждый `git commit` — это запись в бортовом журнале, каждый баг — морское чудовище, а деплой — триумфальное возвращение в порт.

Наш проводник — **Termix**, кибер-пиратский кот с неоновым глазом и чёрной треуголкой. Он знает Python, JavaScript, Spline 3D и умеет объяснить `git rebase` через мем с попугаем.
